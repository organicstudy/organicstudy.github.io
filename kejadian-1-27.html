8. (27)pangkat minus 1 jika diubah menjadi bentuk bilangan berpangkat positif menjadi​


[tex] \frac{1}{27} [/tex]

Pangkat 1 boleh tidak dituliskan, pangkat negatif dapat diubah menjadi positif dengan permisalan jika Ada a pangkat minus m maka jika diubah menjadi 1 per a pangkat m. Seperti soal diatas tetapi karena pangkat 1 boleh tidak dituliskan di 27nya. Contoh bentuknya :

[tex] {a}^{ - m} = \frac{1}{ {a}^{m} } [/tex]

Contoh :

[tex] {12}^{ - 6} [/tex]

Maka menjadi

[tex] \frac{1}{ {12}^{6} } [/tex]

16. hasil dari 3^(2x-1)=11/27 jadi berapa nilai x


3^(2x - 1) = 11/27
3^(2x) /3 = 11/27
3^(2x) = 3 × 11/27
3^2x = 11/9
(3^x)^2 = ((√11)/3)^2
3^x = (√11)/3
x log 3 = log (√11) - log 3
x = 3'log 11^(1/2) - 3'log 3
x = (1/2)(3'log 11) - 1
x = 0,091
