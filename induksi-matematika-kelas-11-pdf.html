1) 1 + 3 + 5 + .... + (2n - 1) = n²

a. Akan dibuktikan untuk n = 1 benar
(2 . 1 - 1) = 1²
1 = 1
BENAR

b. Misal untuk n = k benar
1 + 3 + 5 + .... + (2k - 1) = k²

akan dibuktikan untuk n = (k + 1) juga benar
1 + 3 + 5 + ... + (2k - 1) + (2(k + 1) - 1) = (k + 1)²
k² + (2(k + 1) - 1) = (k + 1)²
k² + 2k + 2 - 1 = (k + 1)²
k² + 2k + 1 = (k + 1)²
(k + 1)² = (k + 1)²
TERBUKTI

2) 5 + 9 + 13 + ... + (4n + 1) = 2n² + 3n
5 + 9 + 13 + ... + (4n + 1) = n(2n + 3)

a. Akan dibuktikan untuk n = 1 benar
(4 . 1 + 1) = 1(2.1 + 3)
(4 + 1) = 1 . (2 + 3)
5 = 5
BENAR

b. Misal untuk n = k benar
5 + 9 + 13 + ... + (4k + 1) = k(2k + 3)

akan dibuktikan untuk n = (k + 1) juga benar
5 + 9 + 13 + ... + (4k + 1) + (4(k + 1) + 1) = (k + 1)(2(k + 1) + 3)
k(2k + 3) + (4(k + 1) + 1) = (k + 1)(2k + 2 + 3)
2k² + 3k + 4k + 4 + 1 = (k + 1)(2k + 5)
2k² + 7k + 5 = (k + 1)(2k + 5)
(k + 1)(2k + 5) = (k + 1)(2k + 5)
TERBUKTI
