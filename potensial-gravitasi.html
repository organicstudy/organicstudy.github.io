13. Jika potensial gravitasi pada permukaan bumi adalah -V,potensial gravitasi pada ketinggian 3R dari permukaan bum kkb adalah....


Potensial gravitasi pada ketinggian 3R dari permukaan bumi adalah:

-¹/₄V

[tex]\texttt{ }[/tex]

PEMBAHASAN

Sebelum menjawab soalnya , marilah kita ingat kembali beberapa rumus penting tentang Gravitasi Newton seperti berikut ini.

Energi potensial gravitasi dari sebuah benda didefinisikan sebagai usaha yang diperlukan untuk memindahkan benda bermassa m tersebut dari jauh tak berhingga ke sebuah titik berjarak r dari benda bermassa M :

Ep = - G . M . m / r

V = - G . M / r

dimana :

Ep = Energi potensial gravitasi ( Joule )

V = Potensial gravitasi ( Joule / kg )

G = Konstanta Gravitasi ( 6,67 x 10⁻¹¹ Nm² / kg² )

m = massa benda ( kg )

r = jarak antar benda ( m )

Okay mari kita gunakan prinsip ini untuk menyelesaikan soalnya.

[tex]\texttt{ }[/tex]

Diketahui :

Potensial Gravitasi Permukaan Bumi = V_A = - V

Jari - Jari Bumi = R

Ketinggian = h = 3R

Ditanyakan :

Potensial Gravitasi pada ketinggian 3R = V_B = ?

Penyelesaian :

Untuk menyelesaikan soal ini kita akan langsung membandingkan potensial gravitasi pada permukaan bumi dengan potensial gravitasi pada ketinggian 3R dari permukaan bumi.

[tex]V_A : V_B = -G \frac{M}{R_A} : -G \frac{M}{R_B}[/tex]

[tex]V_A : V_B = \frac{1}{R_A} : \frac{1}{R_B}[/tex]

[tex]V_A : V_B = R_B : R_A[/tex]

[tex]V_A : V_B = ( R + h ) : R[/tex]

[tex]V_A : V_B = ( R + 3R ) : R[/tex]

[tex]V_A : V_B = 4R : R[/tex]

[tex]V_A : V_B = 4 : 1[/tex]

[tex]- V : V_B = 4 : 1[/tex]

[tex]V_B = \frac{1}{4} (-V)[/tex]

[tex]\boxed{V_B = - \frac{1}{4} V}[/tex]

Kesimpulan :

Potensial gravitasi pada ketinggian 3R dari permukaan bumi adalah:

-¹/₄V
