2. Sebuah tabung memiliki luas permukaan 880 cm². Jika diameter tabung 14 cm, maka tinggi tabung tersebut adalah ... cm​


[MATHEMATICS]

Tinggi tabung tersebut adalah 13 cm

PEMBAHASAN

Tabung adalah bangun ruang tiga dimensi (3D) yang dibentuk oleh dua buah lingkaran yang sejajar dan sama panjang serta sebuah persegi panjang yang mengelilingi kedua lingkaran tersebut yang disebut selimut tabung. Tabung memiliki 3 sisi dan 2 rusuk. Rumus rumus dalam tabung :

Volume Tabung

[tex]\boxed{ πr²t }[/tex]

Luas Permukaan Tabung

[tex]\boxed{ 2πr(r\: +\: t) }[/tex]

Luas Permukaan Tabung Tanpa Tutup

[tex]\boxed{ πr(2t\: +\: r) } [/tex]

Luas Selimut Tabung

[tex]\boxed{ 2πrt }[/tex]

Ket :

π = phi (3,14 atau 22/7)

r = radius atau jari - jari

t = tinggi tabung

Luas permukaan 880 cm²

Diameter 14 cm => r = d : 2 = 14 cm : 2 = 7 cm

.

880 cm² = 2 × 22 (7 cm + t)

880 cm² = 44 (7 cm + t)

880 cm² : 44 = 7 cm + t

20 cm = 7 cm + t

20 cm - 7 cm = t

13 cm = t

.

Tinggi tabung tersebut adalah 13 cm

10. sebuah tabung tanpa tutup memiliki luas selimut 880 cm2 jika diketahui tinggi tabung 10 cm berapakah luas permukaan tabung tersebut ?pakai cara!!


Luas permukaan tabung tersebut adalah  1.496 cm².

Pembahasan

Tabung adalah bangun ruang dengan sisi lengkung

Rumus-rumus yang ada pada bangun tabung

1) Luas alas tabung = π x r²

2) Luas selimut tabung = 2 x π x r x t

 atau = π x d x t

3) Luas permukaan tabung tanpa tutup = luas alas + luas selimut

   = π x r² + 2 x π x r x t

   = π x r x (r + 2t)

4) Luas permukaan tabung dengan tutup =  luas alas + luas tutup + luas selimut

  = π x r² + π x r² + 2 x π x r x t

  = 2 x π x r² + 2 x π x r x t

  = 2 x π x r x (r + t)

5) Volume tabung = π x r² x t atau luas alas x t

r = jari-jari

d = diameter

t = tinggi

Penyelesaian Soal

Diketahui:

Luas selimut tabung tanpa tutup = 880 cm²

Tinggi tabung = 10 cm

Ditanya:

Luas permukaan tabung tanpa tutup

Jawab:

langkah pertama kita cari panjang jari-jari alas tabung terlebih dahulu

2 x π x r x t = Luas selimut tabung 

2 x [tex]\frac{22}{7}[/tex] x r x 10 = 880

[tex]\frac{440}{7}[/tex] x r = 880

r = 880 x [tex]\frac{7}{440}[/tex]  

r = 14 cm

Langkah Selanjutnya kita cari luas permukaan tabung tanpa tutup

   = π x r x (r + 2t)  

   =  [tex]\frac{22}{7}[/tex] x 14 x (14 + 2x10)

   = 22 x 2 x (14 + 20)

   = 44 x 34

   = 1.496 cm²          

Luas permukaan tabung tersebut adalah  1.496 cm².
