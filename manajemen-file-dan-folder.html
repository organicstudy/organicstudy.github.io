10. Jelaskan fasilitas yang digunakan untuk membantu manajemen file dan folder ayo jawab cepat mau dikumpulin ​


Fasilitas yang digunakan untuk membantu manajemen file dan folder dalam sebuah komputer yang menggunakan sistem operasi windows adalah windows explorer. Fasilitas tersebut menyediakan akses kepada seluruh file dan folder yang berada dalam suatu komputer.

Pembahasan

Manajemen file merupakan suatu sistem yang berfungsi untuk mengatur file-file ke dalam folder dalam sebuah komputer. Sistem tersebut juga dapat digunakan untuk menemukan, memindahkan, menyalin, dan tentu saja membuka file. File yang dimaksud sendiri dapat berupa file gambar, audio, video, teks. program, dan sebagainya.

Ada berbagai hal yang dapat dilakukan melalui windows manager. Berikut adalah beberapa di antaranya:

Drag and drop, yaitu fasilitas yang digunakan untuk memindahkan file dari satu folder ke folder lain, memindahkan dan membuat shortcut pada desktop.Copy file, yaitu fasilitas untuk menduplikasi suatu file.Delete file atau delete folder, yaitu fasilitas untuk menghapus suatu file atau folder.Send to, yaitu fasilitas untuk menggandakan suatu file atau folder ke lokasi lain, seperti USB Drive atau desktop.
