Yang membedakan Scratch dengan aplikasi bahasa pemrograman lainnya adalah:

Scratch adalah bahasa pemrograman visual sehingga lebih mudah dan hanya perlu menarik blok kode dari samping, tanpa harus mengetikkan kode.

Scratch bisa menjalankan program tanpa perlu melakukan compile.

Scratch bersifat terbatas dibandingkan dengan bahasa pemrograman lain, seperti Java dan Visual Basic. Misalnya, Scratch tidak dapat berinteraksi dengan file di penyimpanan komputer.

Pembahasan:

Scratch adalah bahasa pemrograman tingkat tinggi yang bersifat visual yang ditargetkan terutama untuk anak-anak berusia 8–16 tahun sebagai alat pendidikan untuk pemrograman. Pengguna bahasa pemrograman ini dapat membuat proyek secara online di situs web Scratch menggunakan antarmuka seperti blok-blok kode yang ditarik, tanpa perlu mengetikkan kode program. Proyek Scratch kemudian dapat diekspor ke HTML5 dan aplikasi Android menggunakan alat eksternal. Layanan ini dikembangkan oleh MIT Media Lab, telah diterjemahkan ke dalam 70-an bahasadi dunia.

Scratch memungkinkan pengguna menggabungkan berbagai media (termasuk grafik, suara, dan program lainnya) dengan cara yang kreatif dengan membuat dan 'mencampur ulang' proyek, seperti video game, animasi, dan simulasi.

Bahasa pemrograman visual ini jauh lebih mudah dengan hanya menyentuh blok lalu mengetikkan detail tambahan untuk membuat cerita, permainan, dan animasi interaktif, tanpa mengetikkan tindakan yang sama dengan Python atau JavaScript.

Namun, karena bersifat visual tanpa kode dan ditujukan untuk pemula, Scratch bersifat terbatas dibanding bahasa pemrograman lain, misalnya tidak dapat berinteraksi dengan file.

Jawaban:

Pemrograman Visual dan Scratch adalah dua jenis pemrograman yang berbeda namun keduanya memiliki fokus pada visualisasi pemrograman yang lebih intuitif dan mudah dipahami oleh pemula.

Pemrograman Visual adalah jenis pemrograman yang memungkinkan pengguna untuk membuat kode dengan menggunakan antarmuka grafis atau visual. Beberapa contoh pemrograman visual termasuk Blockly, App Inventor, dan LabVIEW. Dalam pemrograman visual, kode atau program dibuat dengan menyusun blok-blok grafis atau objek visual, kemudian menghubungkannya untuk membentuk sebuah program. Pemrograman visual memungkinkan pemula atau non-programmer untuk memahami pemrograman tanpa perlu mengetahui bahasa pemrograman khusus.

Scratch adalah salah satu contoh dari pemrograman visual yang populer digunakan di kalangan anak-anak atau pemula. Scratch memungkinkan pengguna untuk membuat program dengan cara menarik dan menjatuhkan blok-blok grafis yang berisi perintah dan operasi pemrograman. Dengan Scratch, pengguna dapat membuat animasi, game, dan aplikasi interaktif dengan mudah.

Berikut ini adalah contoh program sederhana yang dapat dibuat dengan Scratch:

Program Salam

Pertama, buka Scratch dan buat proyek baru.

Klik pada sprite dan buat karakter atau avatar yang ingin dijadikan sebagai pengirim salam.

Tambahkan sebuah blok "when flag clicked" di bawah sprite.

Tambahkan sebuah blok "say Hello, world!" di bawah blok sebelumnya.

Klik pada bendera hijau di sudut kanan atas untuk menjalankan program, karakter akan memberikan ucapan salam.

Program Gerakan

Buka Scratch dan buat proyek baru.

Tambahkan sebuah sprite atau avatar yang ingin digerakkan.

Tambahkan sebuah blok "when flag clicked" di bawah sprite.

Tambahkan beberapa blok "move 10 steps" atau "turn 15 degrees" di bawah blok sebelumnya.

Klik pada bendera hijau di sudut kanan atas untuk menjalankan program, karakter akan bergerak sesuai dengan perintah yang diberikan.

16. Terangkan mengenai program scratch untuk arduino​


Jawaban:

Scratch untuk Arduino merupakan sebuah program visual programming yang memungkinkan pengguna untuk membuat program untuk Arduino dengan menggunakan blok-blok visual. Scratch untuk Arduino didasarkan pada Scratch, sebuah program visual programming yang dikembangkan oleh Lifelong Kindergarten Group di MIT Media Lab.

Dalam Scratch untuk Arduino, pengguna dapat menggunakan blok-blok visual yang disediakan untuk membuat program yang terdiri dari beberapa tindakan atau aksi, seperti membaca atau menulis data ke pin digital atau analog pada Arduino, membaca sensor, dan banyak lagi.

Untuk menggunakan Scratch untuk Arduino, pengguna perlu menginstal plugin khusus pada perangkat lunak Scratch. Plugin ini memungkinkan Scratch untuk berkomunikasi dengan board Arduino dan mengirimkan program yang telah dibuat ke board tersebut.

Setelah plugin diinstal, pengguna dapat memilih board Arduino yang akan digunakan dan memilih port serial yang sesuai. Kemudian, pengguna dapat mulai membuat program dengan menambahkan blok-blok visual dan menghubungkannya bersama-sama. Setelah program selesai dibuat, pengguna dapat mengunggahnya ke board Arduino dan menjalankan program tersebut.

Scratch untuk Arduino merupakan alat yang sangat berguna bagi pemula yang ingin mempelajari pemrograman dan elektronika. Dengan menggunakan Scratch untuk Arduino, pengguna dapat dengan mudah membuat program yang kompleks tanpa harus menguasai bahasa pemrograman yang sulit.

Definisi saya tentang Scratch adalah sebuah aplikasi pemrograman visual yang memudahkan anak-anak dan pemula dalam belajar coding dengan cara menyusun blok kode visual secara logis tanpa perlu menghafal sintaks bahasa pemrograman.

2. Contoh perintah program dari code menu di "Events" pada aplikasi Scratch antara lain:

- When Flag Clicked: menjalankan program ketika bendera di klik

- When Space Key Pressed: menjalankan program ketika tombol spasi ditekan

- When Green Flag Clicked: menjalankan program ketika bendera hijau di klik

- When Sprite Clicked: menjalankan program ketika objek di klik

- When Timer reaches: menjalankan program ketika waktu yang diatur mencapai batas waktu yang ditentukan

3. Contoh perintah program dari code menu di "Control" pada aplikasi Scratch antara lain:

- Repeat: mengulangi sebuah blok kode sebanyak jumlah yang diatur

- Forever: menjalankan blok kode secara terus menerus

- If/Else: menjalankan blok kode jika syarat terpenuhi, dan menjalankan blok kode lain jika syarat tidak terpenuhi

- Wait: menunggu beberapa detik sebelum menjalankan blok kode berikutnya

- Broadcast: mengirim pesan ke objek lain di dalam proyek Scratch

4. Nama perintah untuk menjalankan dan memberhentikan objek/gambar pada aplikasi Scratch adalah "Green Flag" untuk menjalankan objek dan "Stop" untuk memberhentikan objek.
