4. reaksi ionisasi tembaga (ii) hidroksida​


Reaksi ionisasi dari tembaga (II) hidroksida​ adalah Cu(OH)₂ (aq) → Cu²⁺ (aq) + 2OH⁻ (aq)

Pembahasan :

Reaksi ionisasi adalah suatu proses perubahan atom atau kelompok atom netral menjadi atom bermuatan listrik (ion) akibat dari penambahan atau pengurangan elektron dari atom tersebut.

Apabila suatu atom menangkap sejumlah elektron dari atom lain maka atom tersebut akan bermuatan negatif (ion negatif = anion).

Apabila atom tersebut melepaskan sejumlah elektron yang dimilikinya, maka atom tersebut akan bermuatan positif (ion positif = kation).

7. Jika barium hidroksida dianggap terionisasi sempurna,tentukan PH larutan 2 liter barium hidriksida 0,2 M


Jawaban:

pH = 13 + log 4

Penjelasan:

Barium Hidroksida rumus kimianya adalah Ba(OH)2.

Pertama-tama kita buat reaksi ionisasi dari Ba(OH)2 :

Ba(OH)2 ⇒ [tex]Ba^{2+}[/tex] + 2[tex]OH^{-}[/tex]

Lihatlah koefisien dari [tex]OH^{-}[/tex], angka 2 yang ada disana adalah valensi basa. (kita akan memerlukannya untuk menghitung konsentrasi [tex]OH^{-}[/tex]).

Kedua, kita hitung konsentrasi [tex]OH^{-}[/tex]atau [[tex]OH^{-}[/tex]] :

[[tex]OH^{-}[/tex]] = M × valensi basa

[[tex]OH^{-}[/tex]] = 0,2 × 2

[[tex]OH^{-}[/tex]] = 0,4

[[tex]OH^{-}[/tex]] = 4 × [tex]10^{-1}[/tex]

Ketiga, kita hitung pOH dari Ba(OH)2 :

pOH = - log [[tex]OH^{-}[/tex]]

pOH = - log (4 × [tex]10^{-1}[/tex])

pOH = - log 4 - log [tex]10^{-1}[/tex]

pOH = 1 - log 4

Keempat, kita hitung pH :

pH + pOH = 14

pH + 1 - log 4 = 14

pH = 14 - (1 - log 4)

pH = 14 - 1 + log 4

pH = 13 + log 4

Catatan :

Dalam perhitungan pH larutan sederhana seperti ini, volume tidak mempengaruhi pH dan pOH. Volume akan mempengaruhi jika terjadi pencampuran antara dua larutan atau lebih.

26. Persamaan reaksi karbon dioksida dgn barium hidroksida


CO2 + Ba(OH)2 <=> BaCO3 + H2O

Penjelasan

Untuk menentukan hasil reaksi kita dapat menjabarkan senyawa menjadi satuan unsur lalu menggabungkannya lagi menjadi senyawa hasil. Berikut caranya :

CO2 + Ba(OH)2

= C + O + O + Ba + O2 + H2

= C + O + O + Ba + O + O + H2

= C + O + O + Ba + O + H2O

= C + O + O + O + Ba + H2O

= C + O3 + Ba + H2O

= CO3 + Ba + H2O

= BaCO3 + H2O

Jadi, persamaan reaksinya adalah :

CO2 + Ba(OH)2 <=> BaCO3 + H2O
