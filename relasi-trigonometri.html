2. tolong jelaskan sudut relasi trigonometri no 53


Perbandingan trigonometri sudut
cos α = sin (90° - α)

cos (3x + 30°) = sin 2x
sin (90° - (3x + 30°)) = sin 2x
90° - (3x + 30°) = 2x
90° - 3x - 30° = 2x
-3x + 60° = 2x
5x = 60°
x = 60°/5 = 12°

7. Hitunglah nilai perbandingan trigonometri berikut dengan menggunakan relasi (90°+a) dan (180°-a) a. cosec 135°b. tan 120°c. cos 150°​


Jawaban:

Rumus berubah (90+a) :

Sin >> - cos

cosec >> - sec

tan >> - tan

Berlaku juga untuk sebaliknya.

a. cosec 135°

•• menggunakan relasi (90+a)

[tex]cosec \: (90 + 45) = - sec \: 45 = - \frac{2}{ \sqrt{2} } [/tex]

[tex]karena \: sec \: di \: kuadran \: dua \: negatif \: maka \: nilainya \: menjadi \: = - ( - \frac{2}{ \sqrt{2} } ) = \frac{2}{ \sqrt{2} } = \sqrt{2} [/tex]

•• menggunakan relasi (180-a)

cosec 135 = cosec ( 180-45) = cosec 45 = √2

b. tan 120

•• menggunakan relasi (90+a)

[tex]tan \: 120 = tan \: (90 + 30) = -cot \: 30 = - \sqrt{3} [/tex]

•• menggunakan relasi (180-a)

di kuadran dua tan bernilai negatif

-tan 120 = - tan (180-60) = - tan 60 = -√3

c. cos 150

•• menggunakan relasi (90+a)

[tex]cos \: 150 = cos \: (90 + 60) = - sin \: 60 = - \frac{1}{2} \sqrt{3} [/tex]

•• menggunakan relasi (180-a)

di kuadran dua cos bernilai negatif

-cos 150 = -cos (180-30)= - cos 30 = -½√3
