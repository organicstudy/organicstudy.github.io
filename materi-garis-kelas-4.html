11. Tentukan persamaan garis lurus yang melalui titik potong 2x = 3y - 5 dan x = 4y - 1 serta tegak lurus dengan persamaan 3x + ½y + 4 = 0 !Materi: SPLDVKelas: VIIIType soal: HOTSSelamat mengerjakan semua :)​


DIKETAHUI

Garis melalui titik potong [tex]\sf2x=3y-5[/tex] dan [tex]\sf x=4y-1[/tex]

Tegak lurus [tex]\sf3x+\dfrac{1}{2}y+4=0[/tex]

ㅤ

DITANYA

Persamaan garisnya

ㅤ

JAWAB

Menentukan titik potong [tex]\sf2x=3y-5[/tex] dan [tex]\sf x=4y-1[/tex]

[tex]\begin{aligned}\sf2x&=\sf3y-5\\\sf2(4y-1)&=\sf3y-5\\\sf8y-2&=\sf3y-5\\\sf8y-3y&=\sf2-5\\\sf8y-3y&=\sf2-5\\\sf5y&=\sf-3\\\sf y&=\sf-\dfrac{3}{5}\end{aligned}[/tex]

[tex]\begin{aligned}\sf x&=\sf4y-1\\\sf x&=\sf4\left(-\dfrac{3}{5}\right)-\dfrac{5}{5}\\\sf x&=\sf-\dfrac{12}{5}-\dfrac{5}{5}\\\sf x&=\sf-\dfrac{17}{5}\\\end{aligned}[/tex]

[tex]\sf Titik~potongnya~\left(-\dfrac{17}{5},~-\dfrac{3}{5}\right)[/tex]

ㅤ

Menentukan gradien [tex]\sf3x+\dfrac{1}{2}y+4=0[/tex]

[tex]\begin{aligned}\sf3x+\dfrac{1}{2}y+4&=\sf0\\\sf6x+y+8&=\sf0\\\sf y&=\sf-6x-8 \implies m_1=-6\end{aligned}[/tex]

ㅤ

Karena garisnya tegak lurus [tex]\sf3x+\dfrac{1}{2}y+4=0[/tex] maka:

[tex]\begin{aligned}\sf m_1\times m_2&=\sf-1\\\sf-6\times m_2&=\sf-1\\\sf m_2&=\sf\dfrac{1}{6}\\\end{aligned}[/tex]

ㅤ

Sehingga persamaan garisnya adalah:

[tex]\begin{aligned}\sf y-y_1&=\sf m_2(x-x_1)\\\sf y-\left(-\dfrac{3}{5}\right)&=\sf\dfrac{1}{6}\left(x-\left(-\dfrac{17}{5}\right)\right)\\\sf y+\dfrac{3}{5}&=\sf\dfrac{1}{6}\left(x+\dfrac{17}{5}\right)\\\sf y+\dfrac{3}{5}&=\sf\dfrac{1}{6}x+\dfrac{17}{30}\\\sf y&=\sf\dfrac{1}{6}x+\dfrac{17}{30}-\dfrac{3}{5}\\\sf y&=\sf\dfrac{1}{6}x+\dfrac{17}{30}-\dfrac{18}{30}\\\sf y&=\sf\dfrac{1}{6}x-\dfrac{1}{30}\end{aligned}[/tex]

Jawaban:

Penyelesaian :

Menentukan titik potong 2x = 3y dan x = 4y - 1

=> 2x = 3y - 5

=> 2 (4y - 1) = 3y - 5

=> 8y - 2 = 3y - 5

=> 8y - 3y = 2 - 5

=> 5y = -3

=> y = -3/5

=> x = 4y - 1

=> x = 4 (-3/5) - 5/5

=> x = -17/5

Jadi, titik potong = -3/5 dan -17/5

Menentukan gradier 3x + 1/2y + 4 = 0

=> 3y + 1/2y + 4 = 0

=> 6x + y + 8 = 0

=> y = -6x - 8 = m1 = -6

Jika, garis tegak lurusnya = 3x + 1/2y + 4 = 0 berarti ;

=> m1 × m2 = -1

=> -6 × m2 = -1

=> m2 = 1/6

Jadi, persamasn garisnya adalah ;

=> y - y1 = m2 (x - x1)

=> y - (-3/5) = 1/6 (x - (-17/5))

=> y + 3/5 = 1/6 (x + 17/5)

=> y + 3/5 = 1/6x + 17/30

=> y = 1/6x + 17/30 - 3/5

=> y = 1/6x + 17/30 - 18/30

=> y = 1/6x - 1/30

Jawab:

[tex]D. \ 4(\sqrt{6} + \sqrt{3} + 3) \ cm[/tex]

Penjelasan dengan langkah-langkah:

PQ² = QZ x QR

       = 8 x 12 = 96

[tex]PQ = \sqrt{96} = \sqrt{16 \times 6} =4\sqrt{6} \ cm[/tex]

PR² = RZ x RQ

      = 4 x 12

      = 48

[tex]PR = \sqrt{48} = \sqrt{16 \times 3} = 4\sqrt{3} \ cm[/tex]

Keliling Segitiga PQR = PQ + PR + QR

                                   [tex]= 4\sqrt{6} + 4\sqrt{3} + (8 + 4)\\ \\= 4\sqrt{6} + 4\sqrt{3} + 12\\ \\= 4(\sqrt{6} + \sqrt{3} + 3) \ cm[/tex]
