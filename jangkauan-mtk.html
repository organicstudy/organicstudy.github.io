5. jangkauan dan jangkauan interkuental 5,5,7,9,8,10,3,4,5,3​


Jangkauan = 7

Jangkauan interkuartil = 4

Pembahasan

Urutan data dari yg terkecil:

3, 3, 4, 5, 5, 5, 7, 8, 9, 10

Jangkauan suatu data adalah selisih antara data terbesar dan data terkecil.

Data terbesar = 10

Data terkecil = 3

Jangkauan = 10 - 3 = 7

Ukuran yg membagi data menjadi 4 kelompok yg sama disebut kuartil. Ada 3 jenis kuartil, yaitu kuartil bawah (Q1), kuartil tengah (Q2), dan kuartil atas (Q3). Data tersebut merupakan data genap, maka:

Menentukan kuartil bawah:

Q1 = 1/4 (10 + 2)

Q1 = 1/4 (12)

Q1 = 3

Kuartil bawah terletak pada data ke-3, yaitu 4.

Menentukan kuartil atas:

Q3 = 1/4 (3n+ 2)

Q3 = 1/4 (3.10 + 2)

Q3 = 1/4 (30 + 2)

Q3 = 1/4 (32)

Q3 = 8

Kuartil atas terletak pada data ke-8, yaitu 8.

Jangkauan interkuartil adalah selisih antara kuartil atas dan kuartil bawah.

Qr = Q3 - Q1

Qr = 8 - 4

Qr = 4


6. dari tes kompetensi mtk siswa kelas viii smp diperoleh skor sbb 50,60,45,70,95,35,80,65,70,55,65,70,40,35,70,65,50,75,80,40 dari data di atas ,tentukan jangkauan ,kuartil bawah ,kuartil tengah kuartil atas dan jangkauan interkuartil​


Tentukan jangkauan, kuartil bawah, kuartil tengah, kuartil atas, dan jangkauan interkuartil​! Jangkauannya adalah 60, Kuartil bawah adalah 47,5, Kuartil tengah adalah 65, kuartil atas adalah 70, Jangkauan interkuartil adalah 22,5

Pembahasan

Dalam ilmu statistika, kita mengenal ukuran pemusatan data dan ukuran penyebaran data. Beberapa yang termasuk ukuran pemusatan data dan rumusnya antara lain:

1. Rata-rata (mean) = Jumlah seluruh frekuensi/Banyak data

2. Median = Nilai tengah

3. Modus = Nilai paling sering muncul

4. Kuartil

[tex]Q_{i} = x_{\frac{i\times n + 2}{4}}[/tex]

5. Desil

[tex]D_{i} = x_{\frac{i\times n + 1}{10}}[/tex]

Sedangkan, yang termasuk dalam ukuran penyebaran data dan rumusnya antara lain:

1. Range (jangkauan) = datum tertinggi - datum terendah

2. Jangkauan antarkuartil/interkuartil = kuartil atas - kuartil bawah = Q₃ - Q₁

Nah, untuk menyelesaikan persoalan tersebut, langkah pertama yang perlu dilakukan adalah mengurutkan data tersebut terlebih dahulu.

35, 35, 40, 40, 45, 50, 50, 55, 60, 65, 65, 65, 70, 70, 70, 70, 75, 80, 80, 95

1. Mencari jangkauan

Kalau kita lihat di urutan datanya, datum tertinggi adalah 95 dan datum terendah adalah 35, maka

Range = 95 - 35 = 60

2. Mencari Kuartil Bawah (Q₁)

[tex]Q_{i} = x_{\frac{i\times n + 2}{4}}[/tex]

[tex]Q_{1} = x_{\frac{1\times 20 + 2}{4}}[/tex]

[tex]Q_{1} = x_{\frac{20 + 2}{4}}[/tex]

[tex]Q_{1} = x_{\frac{22}{4}}[/tex]

[tex]Q_{1} = x_{5,5}[/tex]

Q₁ = data ke-5 + 1/2 (data ke-6 - data ke-5)

Q₁ = 45 + 1/2(50 - 45)

Q₁ = 45 + 1/2(5)

Q₁ = 45 + 2,5 = 47,5

3. Mencari Kuartil Tengah (Q₂)

[tex]Q_{i} = x_{\frac{i\times n + 2}{4}}[/tex]

[tex]Q_{2} = x_{\frac{2\times 20 + 2}{4}}[/tex]

[tex]Q_{2} = x_{\frac{40 + 2}{4}}[/tex]

[tex]Q_{2} = x_{\frac{42}{4}}[/tex]

[tex]Q_{2} = x_{10,5}[/tex]

Q₂ = data ke-10 + 1/2 (data ke-11 - data ke-10)

Q₂ = 65 + 1/2(65 - 65)

Q₂ = 65 + 1/2(0)

Q₂ = 65

4. Mencari Kuartil Atas (Q₃)

[tex]Q_{i} = x_{\frac{i\times n + 2}{4}}[/tex]

[tex]Q_{3} = x_{\frac{3\times 20 + 2}{4}}[/tex]

[tex]Q_{3} = x_{\frac{60 + 2}{4}}[/tex]

[tex]Q_{3} = x_{\frac{62}{4}}[/tex]

[tex]Q_{3} = x_{15,5}[/tex]

Q₃ = data ke-15 + 1/2 (data ke-16 - data ke-15)

Q₃ = 70 + 1/2(70 - 70)

Q₃ = 70 + 1/2(0)

Q₃ = 70

5. Mencari Jangkauan Interkuartil/Antarkuartil

Jangkauan antarkuartil = Q₃ - Q₁ = 70 - 47,5 = 22,5

Jangkauan (X) = Max (X) - Min (X)

Penyelesaian :

1) urutkan data dari terkecil ke terbesar

5, 6, 7, 7, 8, 9, 9, 10

2) Data maximal = 10

Data Minimal = 5

Maka,

Jangkauan (X) = Max (X) - Min (X)

= 10 - 5

= 5

Jadi, jangkauannya = 5
