22. perhatikan gambar !persamaan garis c pada gambar adalah


Kelas 8 Matematika
Bab Persamaan Garis

Penyelesaian :

• Gradien garis yang sejajar dengan garis c, karena miring ke arah kiri maka gradiennya negatif.
m = - y/x
m = - 4/2
m = - 2

• Karena sejajar, maka m1 = m2

• Titik potong garis c
= 4 x - 2
= - 8 => sebagai koordinat sumbu x
Maka, titik koordinat garis c (- 8, 0)

• Persamaan garis c yang melalui titik (- 8, 0)
y - y1 = m(x - x1)
y - 0 = - 2(x - (- 8))
y = - 2(x + 8)
y = - 2x - 16
y + 2x + 16 = 0

Jadi, persamaan garis c adalah y + 2x + 16 = 0.

Opsi => A
