14. Dua buah persegi panjang A dan B memiliki ukuran yang berbeda. Jika diketahui luas persegi panjang A adalah 288cm² dan panjang sisinya 18cm². Sedangkan persegi panjang B memiliki ukuran panjang 15cm dan lebarnya 2 kali dari lebar persegi panjang A. Maka hitunglah gabungan jumlah luas persegi panjang A dan B? TOLONG DONG BANTUANNYA


Bangun datar adalah suatu bidang datar yang tersusun oleh titik atau garis-garis yang menyatu membentuk bangun 2 dimensi yang mempunyai keliling dan luas. Contohnya: persegi panjang, persegi, segitiga, jajar genjang, belah ketupat, layang-layang, trapesium, dan lingkaran.

Persegi panjang adalah bangun datar segi empat yang keempat sudutnya siku-siku dan sisi-sisi yang berhadapan sama panjang.

Rumus :

L = p x l

Keterangan Rumus :

L (Luas), p (panjang), dan l (lebar)

Diketahui :

L persegi panjang A = 288 cm²

p persegi panjang A = 18 cm

p persegi panjang B = 15 cm

l persegi panjang = 2x lebar persegi panjang A

Ditanya :

Berapa Luas gabungan persegi panjang?

Pembahasan :

Langkah pertama, mencari lebar pada persegi panjang A :

[tex]Lpersegi \ panjang \ A \ = \ p \ x \ l\\288 \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ = \ 18 \ x \ l\\\frac{288}{18} \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ = \ l\\16 \ cm \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ = \ l[/tex]

Langkah kedua, mencari lebar persegi panjang B :

[tex]lpersegi \ panjang \ B \ = \ 2 \ x \ lpersegi \ panjang \ A\\lpersegi \ panjang \ B \ = \ 2 \ x \ 16\\lpersegi \ panjang \ B \ = \ 32 \ cm[/tex]

Langkah ketiga, mencari Luas persegi panjang B :

[tex]Lpersegi \ panjang \ B \ = \ p \ x \ l\\Lpersegi \ panjang \ B \ = \ 15 \ x \ 32\\Lpersegi \ panjang \ B \ = 480 \ cm^{2}[/tex]

Langkah terakhir, menghitung Luas gabungan persegi panjang A dan B :

[tex]Lpersegi \ panjang \ A \ + \ Lpersegi \ panjang \ B\\= \ 288 \ cm^{2} \ + \ 480 \ cm^{2} \\= \ 768 \ cm^{2}[/tex]

Kesimpulan :

Jadi, Luas gabungan persegi panjang A dan B adalah 768 cm².
