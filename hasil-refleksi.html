11. Titik Q (-2,5) direfleksikan terhadap sumbu Y. Tentukan hasil refleksinya!


~ Transformasi

♦ Soal:

Titik Q (–2, 5) direfleksikan terhadap sumbu y. Tentukan hasil refleksinya!

♦ Penyelesaian:

Rumus refleksi terhadap sumbu y:

[tex]\boxed{\bold{P \: (a, \: b) \to P' \: (-a, \: b)}}[/tex]

Maka,

P (a, b)

→ P' (–a, b)

.

Q (–2, 5)

→ Q' (–(–2), 5)

→ Q' (2, 5)

♦ Kesimpulan:

Jadi, hasil refleksinya adalah Q' (2, 5).

P(a,b) => P'(-a,b)

P (a, b)

→ P' (–a, b)

Maka, ...

Q' (-(-2), 5)

Q' (2, 5)

13. Titik C(1,5) direfleksikan terhadap sumbu x = -1. Hasil refleksinya adalah​


Jawaban:

Penjelasan dengan langkah-langkah:

C(1,5)=C'(2h-1,5)=C'(2(-1)-1,5)=C'(-2-1,5)=C'(-3,5).

Penjelasan dengan langkah-langkah:

Titik C(1, 5) direfleksikan terhadap sumbu x = -1. Hasil refleksinya adalah

Jawab :

x = 1, y = 5, dan h = -1

C'(x', y') = C'(2h - x, y)

C'(x', y') = C'(2(-1) - 1, 5)

C'(x', y') = C'(-2 - 1, 5)

C'(x', y') = C'(-3, 5)

Jadi, hasil refleksinya adalah C'(-3, 5)

27. hasil refleksi titik B(-3,5) jika di refleksikan terhadap garis y=2 adalah?​


Jawaban sama penjelasan:

garis refleksi adalah y=2, sehingga m=0 (karena garis tersebut horizontal) dan c=2 (konstanta dari persamaan garis).

Rumus refleksi terhadap garis y=mx+c adalah:

(x', y') = (x, -y + 2c - m*x)

Menggantikan nilai-nilai yang diketahui:

x = -3 (koordinat x titik B)

y = 5 (koordinat y titik B)

m = 0 (gradien garis refleksi y=2)

c = 2 (konstanta dari persamaan garis y=2)

(x', y') = (-3, -5 + 2(2) - 0(-3))

Sekarang kita bisa menghitung nilai x' dan y' menggunakan rumus di atas:

x' = -3

y' = -5 + 4 + 0 = -1

Jadi, hasil refleksi titik B(-3,5) jika direfleksikan terhadap garis y=2 adalah (-3,-1)
