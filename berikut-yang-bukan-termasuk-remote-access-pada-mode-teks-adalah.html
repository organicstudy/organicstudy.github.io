2. manfaat remote access


adalah kemampuan untuk terhubung dengan resource pada suatunetwork sentral dari suatu lokasi. Ini berarti menggunakan sebuah PC dan modem di satutempat, lewat kabel telepon, terhubung ke suatu PC atau server pada network utama suatuperusahaan.

3. kekurangan dari remote access


Kekurangan dari remote access adalah harus terhubung dengan jaringan terlebih dahulu. Selain itu, komputer yang akan diremote juga harus sudah menyala terlebih dahulu dan ketika sedang diremote, komputer tidak bisa digunakan terlebih dahulu. Remote access atau dikenal dengan remote desktop adalah suatu fitur yang dimiliki oleh suatu sistem operasi yang memungkinkan penggunanya untuk mengakses komputer lain melalui komputernya.

Remote access dapat dilakukan dengan aplikasi bawaan sistem operasi atau menggunakan aplikasi remote access lainnya seperti teamviewer. Beberapa fungsi remote access secara umum adalah sebagai berikut

1. Dapat mengendalikan komputer lain dari jarak jauh

2. Dapat mengetahui aktivitas apa saja yang ada pada komputer yang diremote.

3. Membantu user lain dalam menyelesaikan masalah komputernya tanpa harus datang ke lokasi.

Demikian pembahasan mengenai remote access, remote access dikenal juga dengan remote desktop, dengan menggunakan remote access, seseorang dapat mengakses atau menggunakan komputer lain tanpa harus datang langsung dan mengakses secara langsung komputer tersebut.

5. 5. Bagaimanakah indikasi jika Routing dan Remote access sudah aktif?​


Jawaban:

Untuk mengetahui apakah Routing and Remote Access (RRAS) sudah aktif atau belum, ada beberapa indikator yang dapat digunakan.

Melalui Server Manager: Buka Server Manager, klik Local Server, lalu klik Remote Access. Jika RRAS sudah aktif, maka akan terlihat tulisan "Remote Access is currently enabled"

Melalui Control Panel: Buka Control Panel, klik Administrative Tools, lalu klik Routing and Remote Access. Jika RRAS sudah aktif, maka akan terlihat tulisan "Routing and Remote Access" di jendela yang muncul.

Melalui Command Prompt: Buka Command Prompt, ketik "RRAS" lalu enter. Jika RRAS sudah aktif, maka akan terlihat tulisan "Routing and Remote Access Service"

Melalui Services: Buka Services, cari Routing and Remote Access, jika statusnya Running, itu menandakan RRAS sudah aktif.

Semua indikator di atas menunjukkan bahwa RRAS sudah aktif dan dapat digunakan untuk routing dan remote access.

14. apa itu vlan? dan jelaskan apa itu vlan mode trunk dan vlan mode access?


Jawaban:

VLAN (Virtual Local Area Network) adalah sekelompok perangkat pada satu LAN atau lebih yang dikonfigurasikan (menggunakan perangkat lunak pengelolaan) sehingga dapat berkomunikasi seperti halnya bila perangkat tersebut terhubung ke jalur yang sama, padahal sebenarnya perangkat tersebut berada pada sejumlah segmen LAN yang berbeda.

VLAN mode trunk adalah port yang dikonfigurasi untuk dilalui berbagai vlan.

VLAN mode access adalah port yang dikonfigurasi hanya untuk satu vlan pada switch tersebut, inti dari mode access yaitu satu port yang bisa terdaftar di satu vlan dan tidak bisa didaftarkan lebih dari satu vlan.

30. Jelaskan beberapa mode pada access point


Jelaskan beberapa mode pada access pointJawabanPendahuluan

Sebelum membahas lebih lanjut mengenai mode pada access point, mari kita bahas terlebih dahulu apa itu access point. Access point adalah salah satu perangkat untuk jaringan komputer wireless atau tanpa kabel.

Pembahasan

Beberapa macam mode pada access point adalah sebagai berikut

1. Bridge Mode, mode ini digunakan untuk menyalurkan kembali suatu jaringan, namun dengan nama dan password yang berbeda.

2. Repeater mode, seperti dengan namanya, mode ini berfungsi untuk mengulang jaringan yang ada dengan tujuan agar sinyal dari jaringan yang diulang dapat lebih kuat dan mampu diterima dengan baik oleh klien.

3. Wireless Router Mode, pada mode ini access point berfungsi untuk membagikan internet dari suatu ISP, mode ini yang biasa dipakai oleh ISP Telkom dan lainnya.

Kesimpulan

Demikian pembahasan mengenai beberapa macam mode pada acces point, mulai dari Wireless Router Mode, Repeater Mode dan Bridge Mode. Access point sendiri merupakan salah satu perangkat pada jaringan komputer wireless. Untuk mempelajari materi lebih lanjut mengenai jaringan komputer wireless dapat dibaca pada link berikut
