6. contoh soal dan pembahasan jarak titik ke garis pada kubus ?


contoh soal
1.) Pada kubus ABCD.EFGH diketahui panjang sisi 10 . Jarak C ke diagonal ruang AG .... ( gambar 1)
2.) pada sebuah kubus ABCD.EFGH  dengan rusuk 20 cm diketahui titik k berada di tegah garis GC tentukan jarak k ke garis DB ..... ( gambar 2)
jawaban nomer 1
ac = diagonal sisi
     = s√2
     = 10√2
ag = diagonal ruang
     = s √3
     = 10√3
gc = rusuk
     = 10
jarak c ke garis AG ( garis merah)
jarak = AC x GC / AG
         = 10√2 X 10 / 10√3
         = 100√2 / 10√3
         = 10√2/√3 rasionalkan
         = 10√2/√3 x √3/√3
         = 10√6/3 cm
jawaban soal nomer 2
ck = 1/2 x cg
     = 1/2 x 20
     = 10
titik tengah sisi ABCD  kita beri simbol O maka
OB = diagonal sisi/2
       = s√2/2
       = 20√2/2
       = 10√2 cm
Bk = Bd
     = √ ( 20² +10²)
     = √ ( 400 + 100)
     = √ 500
     = 10√5 cm
jatak titik k ke garis DB
jarak² = (bk²) - (ob)²
jarak² = (10√5)² - (10√2)²
jarak² = ( 100√25 - 100√4)
jarak² = 500 - 200
jarak² = 300
 jarak = √ 300
 jarak = 10√3 cm

11. pada kubus ABCD EFGH, jika titik P ditengah garis AB dan panjang rusuk kubus 6cm, maka jarak titik P ke garis DE adalah...cm


AB = 6 cm maka AP = 3 cm

Misal Q adalah titik tengah DE
DE² = AD² + AE²
DE² = 6² + 6²
DE² = 2.6²
DE  = √(2.6²)
DE  = 6√2 cm
EQ = 1/2 x 6√2 = 3√2 cm

PE² = AP² + AE²
PE² = 3² + 6²
PE² = 9 + 36
PE² = 45
PE  =√45
PE  = 3√5

Jarak P ke garis DE adalah PQ
PQ² = PE² - EQ²
PQ² = (3√5)² - (3√2)²
PQ² = 45 - 18
PQ² = 27
PQ  = √27
PQ  = 3√3 cm

Jadi, jarak titik P ke garis DE adalah 3√3 cm

29. Diketahui kubus ABCD.EFGH dengan rusuk 6cm. Hitunglah jarak titik D ke garis CH? Jarak titik F ke garis HB?


Materi : Dimensi 3
Bahasan : Jarak antara titik dan garis
Jawaban : D ke CH = 3√2 Cm dan F ke HB = 2√6 Cm

Langkah - langkah :
Jarak titik D ke garis CH = 1/2DG
⇒ DG = 6√2 Cm
⇒ 1/2DG = 3√2 Cm

Jarak titik F ke garis HB
HB merupakan diagonal ruang
⇒ HB = 6√3 Cm
Buat garis HF, sehingga akan berbentuk segitiga siku - siku di F
Tarik garis dari titik F ke garis HB dan beri nama F'. Maka :
FF' = (6√2 x 6) / 6√3
FF' = (36√2) / 6√3
FF' = (6√2) / √3
(Rasionalkan)
FF' = (6√6) / 3
FF' = 2√6 Cm
