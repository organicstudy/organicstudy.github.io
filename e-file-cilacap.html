12. 1. Jelaskan pengertian beberapa jenis file dibawah ini berdasarkan ekstensinya:File Sistemb. File VideoC. File Dokumend. File Gambare. File Suaraf. File Archive​


Jawaban:

A. Pengertian file sistem : struktur logika yang digunakan untuk. mengendalikan akses terhadap data yang ada pada harddisk. q

B. Pengertian dari file video : berkas komputer yang digunakan untuk menyimpan kumpulan berkas digital seperti video, audio, metadata, informasi, pembagian chapter, dan judul sekaligus, yang dapat dimainkan atau digunakan melalui perangkat lunak tertentu pada komputer.

C. Pengertian dari file dokumen : File yang mengandung informasi tertentu dan dapat dibuka dengan program.

D. Pengertian dari file gambar : cara terstandarisasi untuk mengatur dan menyimpan gambar digital.

E. Pengertian dari file suara : jenis file yang merekam atau menyajikan bunyi, bisa berupa lagu, rekaman radio, rekaman telepon dan lain sebagainya.

F. Pengertian dari file archive : tipe file yang terdiri dari file-file lain yang sudah dimampatkan ukurannya (Compressed).

23. sebutkan penjelasan dengan beserta contohnya mp3 dan mkva. file mp3b. file mkvc. file mp4d. file aplikasie. file dokumen ​


JAWABAN+PENJELASAN:OPSI (A). File MP3 adalah format audio yang populer untuk menyimpan musik atau suara digital. Contohnya: file musik atau podcast.

OPSI (B). File MKV adalah format video yang dapat menyimpan audio, video, dan subtitle dalam satu file. Contohnya: file film atau serial TV.

OPSI (C). File MP4 adalah format video yang umum digunakan untuk diunggah ke media sosial dan layanan streaming. Contohnya: video musik atau tutorial.

OPSI (D). File aplikasi adalah file yang berisi program atau software untuk diinstal di perangkat. Contohnya: file installer game atau software.

OPSI (E). File dokumen adalah format file yang digunakan untuk menyimpan teks atau informasi dalam bentuk digital. Contohnya: file Microsoft Word, PDF, atau Google Docs.By: frostfiree69NOTE:SEMOGA MEMBANTU!! ^_^
