Jawaban:

jawaban ada di lampiran ya

a.

2 sin² x - 1 = 0 , untuk 0 ≤ x ≤ 2π

« INGAT : p² - q² = (p - q)(p + q) »

2 sin² x - 1 = 0

(√2 sin x - 1)(√2 sin x + 1) = 0

(√2 sin x - 1) = 0 atau (√2 sin x + 1) = 0

√2 sin x = 1 atau √2 sin x = –1

sin x = ½√2 atau sin x = –½√2

¤ Untuk sin x = ½√2 :

x = ¹/₄π atau x = ³/₄π

¤ Untuk sin x = –½√2 :

x = ⁵/₄π atau x = ⁷/₄π

Jadi HP = { ¹/₄π , ³/₄π , ⁵/₄π , ⁷/₄π }

b.

2 cos² x - 5 cos x - 3 = 0 , untuk 0 ≤ x ≤ 2π

(cos x - 3)(2 cos x + 1) = 0

(cos x - 3) = 0 atau (2 cos x + 1) = 0

cos x = 3 atau 2 cos x = –1

cos x = 3 atau cos x = –½

¤ Untuk cos x = 3 :

Karena nilai maksimum fungsi cos adalah +1, maka untuk cos x = 3 tidak ada nilai x yang memenuhi.

¤ Untuk cos x = –½ :

x = ²/₃π atau x = ⁴/₃π

Jadi HP = { ²/₃π , ⁴/₃π }

c.

tan² 3x - 1 = 0 , untuk 0 ≤ x ≤ 2π

(tan 3x - 1)(tan 3x + 1) = 0

(tan 3x - 1) = 0 atau (tan 3x + 1) = 0

tan 3x = 1 atau tan 3x = –1

¤ Untuk tan 3x = 1 :

3x = ¹/₄π atau 3x = ⁵/₄π

» untuk 3x = ¹/₄π :

3x = ¹/₄π + (k).2π

x = ¹/₁₂π + (k).²/₃π

• k = 0 : x = ¹/₁₂π + (0).²/₃π = ¹/₁₂π + 0 = ¹/₁₂π

• k = 1 : x = ¹/₁₂π + (1).²/₃π = ¹/₁₂π + ²/₃π = ³/₄π

• k = 2 : x = ¹/₁₂π + (2).²/₃π = ¹/₁₂π + ⁴/₃π = ¹⁷/₁₂π

» untuk 3x = ⁵/₄π :

3x = ⁵/₄π + (k).2π

x = ⁵/₁₂π + (k).²/₃π

• k = 0 : x = ⁵/₁₂π + (0).²/₃π = ⁵/₁₂π + 0 = ⁵/₁₂π

• k = 1 : x = ⁵/₁₂π + (1).²/₃π = ⁵/₁₂π + ²/₃π = ¹³/₁₂π

• k = 2 : x = ⁵/₁₂π + (2).²/₃π = ⁵/₁₂π + ⁴/₃π = ²¹/₁₂π

¤ Untuk tan 3x = –1 :

3x = ³/₄π atau 3x = ⁷/₄π

» untuk 3x = ³/₄π :

3x = ³/₄π + (k).2π

x = ¹/₄π + (k).²/₃π

• k = 0 : x = ¹/₄π + (0).²/₃π = ¹/₄π + 0 = ¹/₄π

• k = 1 : x = ¹/₄π + (1).²/₃π = ¹/₄π + ²/₃π = ¹¹/₁₂π

• k = 2 : x = ¹/₄π + (2).²/₃π = ¹/₄π + ⁴/₃π = ¹⁹/₁₂π

» untuk 3x = ⁷/₄π :

3x = ⁷/₄π + (k).2π

x = ⁷/₁₂π + (k).²/₃π

• k = 0 : x = ⁷/₁₂π + (0).²/₃π = ⁷/₁₂π + 0 = ⁷/₁₂π

• k = 1 : x = ⁷/₁₂π + (1).²/₃π = ⁷/₁₂π + ²/₃π = ¹⁵/₁₂π

• k = 2 : x = ⁷/₁₂π + (2).²/₃π = ⁷/₁₂π + ⁴/₃π = ²³/₁₂π

Jadi HP = { ¹/₄π , ¹/₁₂π , ⁵/₁₂π , ⁷/₁₂π , ³/₄π , ¹¹/₁₂π , ¹³/₁₂π , ¹⁵/₁₂π , ¹⁷/₁₂π , ¹⁹/₁₂π , ²¹/₁₂π , ²³/₁₂π }

[tex]{\lim_{x \to 2} \frac{3 x^{2} - 4 x - 4}{x \sin{\left(x - 2 \right)}}} = {\lim_{x \to 2} \frac{1}{x} \lim_{x \to 2} \frac{3 x^{2} - 4 x - 4}{\sin{\left(x - 2 \right)}}}[/tex]

[tex]\lim_{x \to 2} \frac{3 x^{2} - 4 x - 4}{\sin{\left(x - 2 \right)}} \color{red}{\lim_{x \to 2} \frac{1}{x}} = \lim_{x \to 2} \frac{3 x^{2} - 4 x - 4}{\sin{\left(x - 2 \right)}} \color{red}{\frac{\lim_{x \to 2} 1}{\lim_{x \to 2} x}}[/tex]

[tex]\frac{\lim_{x \to 2} \frac{3 x^{2} - 4 x - 4}{\sin{\left(x - 2 \right)}} \color{red}{\lim_{x \to 2} 1}}{\lim_{x \to 2} x} = \frac{\lim_{x \to 2} \frac{3 x^{2} - 4 x - 4}{\sin{\left(x - 2 \right)}} \color{red}{1}}{\lim_{x \to 2} x}[/tex]

[tex]\lim_{x \to 2} \frac{3 x^{2} - 4 x - 4}{\sin{\left(x - 2 \right)}} \color{red}{\lim_{x \to 2} x}^{-1} = \lim_{x \to 2} \frac{3 x^{2} - 4 x - 4}{\sin{\left(x - 2 \right)}} \color{red}{\left(2\right)}^{-1}[/tex]

[tex]\frac{\color{red}{\lim_{x \to 2} \frac{\frac{d}{dx}\left(3 x^{2} - 4 x - 4\right)}{\frac{d}{dx}\left(\sin{\left(x - 2 \right)}\right)}}}{2} = \frac{\color{red}{\lim_{x \to 2} \frac{6 x - 4}{\cos{\left(x - 2 \right)}}}}{2}[/tex]

[tex]=4[/tex]
