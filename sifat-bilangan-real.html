2. sifat sifat bilangan real​


Tertutup : a + b = adalah bilangan real a × b = adalah bilangan real

Asosiatif :: a + (b + c) = (a + b) + c a × (b × c) = (a × b) × c

Komutatif : a + b = b + a a × b = b × a

Mempunyai unsur identitas a + 0 = a a × 1 = a

Setiap bilangan punya invers a + (−a) = 0 a × (1/a) = 1, dengan a ≠ 0

Distributif : a × (b + c) = (a × b) + (a × c)

Tidak ada pembagi nol – jika a × b = 0, maka a = 0 atau b = 0 (atau keduanya)

15. Sebutkan dan jelaskan 2 sifat yang berlaku pada bilangan real


2 sifat yang berlaku pada bilangan real yaitu:

1. Aksioma medan

Bilangan real, beserta operasi penjumlahan dan perkalian, memenuhi aksioma (dianggap terbukti) berikut.  Himpunan yang memenuhi sifat-sifat ini disebut sebagai medan.

2. Aksioma urutan

Contoh terdapat himpunan R+, yang disebut sebagai bilangan positif yang merupakan himpunan bagian dari R. Misalkan juga x dan y adalah anggota R+. Himpunan bagian ini memenuhi aksioma urutan berikut:

• Aksioma 7: x+y dan xy merupakan anggota R+

• Aksioma 8: Untuk setiap x yang tidak sama dengan 0, x anggota R+ atau -x anggota R+, tetapi tidak mungkin keduanya sekaligus

• Aksioma 9: 0 bukan anggota R+.

Pembahasan

Berbeda dengan sifat bilangan real. Berikut adalah sifat-sifat operasi bilangan real. Sifat-sifat yang berlaku pada operasi bilangan real adalah “penjumlahan” dan “perkalian” yaitu:

• Penjumlahan:

a. Sifat tertutup pada penjumlahan

b. Sifat komutatif pada penjumlahan

c. Sifat asosiatif pada penjumlahan

d. Sifat distributif perkalian terhadap penjumlahan

e. Sifat identitas pada penjumlahan (0 adalah elemen identitas atau elemen netral)

f. Sifat invers pada penjumlahan

• Perkalian:

a. Sifat tertutup pada perkalian

b. Sifat komutatif pada perkalian

c. Sifat asosiatif pada perkalian

d. Sifat distributif perkalian terhadap penjumlahan

e. Sifat identitas pada perkalian (1 adalah elemen identitas perkalian)

f. Sifat invers pada perkalian tidak berlaku, sebab 0 tidak mempunyai invers.

24. 1. Untuk semua real x dan y yang tidak sama dengan 0, jika x


Jawaban:

Dari persamaan yang diberikan:

x / y + y / x = 2

Kita bisa mengalikan kedua sisi dengan xy untuk mendapatkan:

x^2 + y^2 = 2xy

Selanjutnya, kita bisa membagi kedua sisi dengan xy untuk mendapatkan:

x^2 / xy + y^2 / xy = 2

Dalam bentuk yang lebih sederhana:

x/y + y/x = 2 -> x^2 + y^2 = 2xy -> x^2/xy + y^2/xy = 2

Kita bisa menyederhanakan menjadi:

x/y + y/x = 2

x^2 + y^2 = 2xy

x^2 + y^2 - 2xy = 0

(x - y)^2 = 0

Karena (x - y)^2 = 0, maka x - y = 0, atau dengan kata lain x = y.

Sehingga, jawaban dari persamaan tersebut adalah x = y.

28. apa yang dimaksud dengan bilangan real rasional dan bilangan real irrasional


Bilangan real rasional adalah bilangan yang dapat dinyatakan dalam a/b, dan a,b adalah rasional.
