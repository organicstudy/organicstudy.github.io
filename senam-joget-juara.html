Karena menentukan susunan juara, maka menggunakan rumus permutasi:

nPr = n! / (n - r)!

n = banyaknya peserta = 12

r = banyaknya juara yg dipilih = 3

12P3 = 12! / (12 - 3)!

= 12! / 9!

= 12.11.10.9! / 9!

= 12.11.10

= 1320 cara
