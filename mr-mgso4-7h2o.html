1. Tentukan Mr dari senyawa MgSO4 . 7H2O


MgSO4.7H2O tersusun atas : 1atom Mg, 1 atom S, 11 atom O dan 14 atom H Mr MgSO4.7H2O = (1 x Ar Mg) + (1 x Ar S) + (11 x Ar O) + (14 x Ar H) = (1 x 24) + (1 x 32) + (11 x 16) + (14 x 1) = 246Mr MgSO4 = (1 x Ar Mg) + (1 x Ar S) + (4 x Ar O)
= (1 x 24) + (1 x 32) + (4 x 16)
= 24 + 32 + 64
= 120
Mr 7H2O = (7 x Ar H) + (2 x Ar O)
= (14 x 1) + (7 x 16)
= 14 + 112
= 126

Mr MgSO4 . 7H2O = Mr MgSO4 + Mr 7H2O
= 120 + 126
= 246

13. Tentukan massa dari 2 mol senyawa MgSO4 jika Mr MgSO4 = 144


Massa dari 2 mol senyawa MgSO₄ (Mr MgSO4 = 144 g/mol) adalah sebanyak 288 gram.

Pembahasan :

Diketahui : mol MgSO₄ = 2 mol

Ditanya : massa ?

Penyelesaian soal :

Untuk menghitung massa dapat dengan menggunakan rumus konsep mol sebagai berikut :

n = [tex]\frac{g}{Ar/ Mr}[/tex]  

Keterangan :  

n = mol  

g = massa (gram)  

Ar = Massa atom relatif (g/mol)  

Mr = Massa molekul relatif (g/mol)  

Secara perhitungan berdasarkan soal diatas maka :

massa MgSO₄ = mol MgSO₄ × Mr MgSO₄ = 2 mol × 144 g/mol = 288 gram

⇒ Kesimpulan, maka massa dari 2 mol senyawa MgSO₄ (Mr MgSO4 = 144 g/mol) adalah sebanyak 288 gram.

19. Berapa Gram kristal MgSO4-7H2O yg harus dilarutkan kedalam air agar diperoleh100 ml MgSO4 0,2 ? ( Ar Mg= 24,0=16 dan H=1)


Berapa Gram kristal MgSO₄.7H₂O yg harus dilarutkan kedalam air agar diperoleh 100 ml MgSO₄ 0,2 ? ( Ar Mg = 24, O = 16 dan H = 1)

Diketahui:

Volume MgSO₄ = 100 mL = 0,1 L

M MgSO₄ = 0,2 M

Ar Mg = 24, O = 16, S = 32, dan H = 1

Ditanya: Massa kristal MgSO₄.7H₂O ......?

Jawab:

Kristal atau garam Hidrat MgSO₄.7H₂O merupakan sebuah kristal dengan kandungan garam dan air, dimana reaksi kimianya bisa kita tuliskan sebagai berikut:

MgSO₄.7H₂O ==> MgSO₄ + 7 H₂O

Jadi kristal tersebut terbentuk dari ikatan antara 1 mol MgSO₄ dengan 7 mol air.

untuk menemukan massa kristal kita bisa menghitung berdasarkan mol MgSO₄, dimana nilai molnya bisa kita hitung berdasarkan molaritas dan volume yang telah diketahui, yaitu:

Molaritas = mol / volume (L)

0,2 M = mol / 0,1 L

mol = 0,02 mol

Maka berdasarkan perbandingan koefisien reaksi antara MgSO₄ dan kristal tersebut, kita bisa menghitung massa kristal yang bereaksi, yaitu:

mol kristal = (Koefisien MgSO₄.7H₂O / Koefisien MgSO₄) x mol MgSO₄

mol Kristal = (1/1) x 0,02 mol

mol Kristal = 0,02 mol

maka berdasarkan mol diatas, massa kristal bisa kita hitung, yaitu:

mol = Massa / Mr

Mr MgSO₄.7H₂O = Ar Mg + Ar S + 4 x Ar O + 7 x (2 x Ar H + Ar O)

Mr MgSO₄.7H₂O = 24 + 32 + 4 x 16 + 7 x (2 x 1 + 16)

Mr MgSO₄.7H₂O = 56 + 64 + 7 x 18

Mr MgSO₄.7H₂O = 120 + 126

Mr MgSO₄.7H₂O = 246

maka massanya

0,02 mol = massa / 246

massa = 4,92 gram

jadi massa kristal MgSO₄.7H₂O yg harus dilarutkan kedalam air agar diperoleh 100 ml MgSO₄ 0,2 M adalah sebesar 4,92 gram.

22. Tentukan Mr dari senyawa ZnSO4 . 7H2O


Mr ZnSO₄. 7H₂O adalah 287

Pembahasan

[tex]\boxed{\boxed{\bold{Stoikiometri}}}[/tex]  

Dalam bahasa Yunani, kata stoicheion berarti unsur sedangkan stoikiometri (stoichiometry) berarti mengukur unsur meliputi massa atom dan massa rumus,kadar, konsentrasi, rumus empiris, rumus molekul, persamaan kimia serta semua yang berhubungan dengan reaksi kimia. Dalam stoikiometri, satuan yang digunakan untuk menyatakan jumlah partikel dalam zat dinamakan mol.  

[tex]\boxed{\boxed{\bold{Diket}}}[/tex]  

ZnSO₄. 7H₂O

[tex]\boxed{\boxed{\bold{Ditanya}}}[/tex]  

Mr

[tex]\boxed{\boxed{\bold{Jawab}}}[/tex]  

Mr (Massa Molekul Relatif)

Massa molekul relatif  (Mr) adalah jumlah total dari massa atom relatif unsur-unsur penyusunnya dalam suatu senyawa atau molekul. Massa molekul relatif (Mr) digunakan untuk sneyawa atau molekul sedangkan massa atom relatif digunakan untuk unsur atau atom. Rumus untuk menentukan massa molekul relatif (Mr) adalah:

[tex]\boxed{\boxed{\bold{Mr~A_nB_m~=~n.~Ar~A~+~m.~Ar~B}}}[/tex]

Massa atom relatif setiap unsur penyusun dari senyawa dapat dilihat pada tabel sistem periodik unsur.

Ar H = 1

Ar N = 14

Ar O = 16

Ar C = 12

Ar S = 32

Ar Zn = 65

Mr ZnSO₄. 7H₂O = Mr ZnSO₄ + Mr 7 (H₂O)

Mr ZnSO₄. 7H₂O = { ( 1. Ar Zn ) + ( 1. Ar S ) + ( 4. Ar O ) } + 7. { ( 2. Ar H ) + ( 1. Ar O ) }

Mr ZnSO₄. 7H₂O = { ( 1. 65 ) + ( 1. 32 ) + ( 4. 16 ) } + 7. { ( 2. 1 ) + ( 1. 16 ) }

Mr ZnSO₄. 7H₂O = { 65 + 32 + 64 } + 7. { 2 + 16 }

Mr ZnSO₄. 7H₂O = { 161 } + 7. { 18 }

Mr ZnSO₄. 7H₂O = 161 + 126

Mr ZnSO₄. 7H₂O = 287

[tex]\boxed{\boxed{\bold{Kesimpulan}}}[/tex]  

Mr ZnSO₄. 7H₂O = 287
