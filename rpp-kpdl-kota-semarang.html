7. Jarak kota semarang ke kota jakarta 600km,sedangkan jarak kota semarang ke kota jakarta 60cm,hitunglah skala


Jawaban :

1 : 1.000.000

Penjelasan : → Diketahui : Jarak Pada Peta ( JP ) = 60 cm Jarak Sebenarnya ( JS ) = 600 km

→ Ditanya :

Skala ( S ) = ???

→ Penyelesaian :

S = JP : JS                                

= 60 cm : 600 km    

= 60 cm : ( 600 × 100.000 ) cm      

= 60 cm : 60.00.000 cm            

= 1 : 1.000.000

Jadi, skala pada peta tersebut adalah 1 : 1.000.000

12. Jarak kota Yogyakarta ke Semarang pada peta berskala 1:500.000 adalah 24cm.jaraj kota Yogyakarta ke kota Semarang sebenarnya adalah


Jawaban:

Penyelesaian :

Jarak kota Yogyakarta ke Semarang pada peta berskala 1 : 500.000 adalah 24 cm. Jarak kota Yogyakarta ke kota Semarang sebenarnya adalah ...

Skala peta = 1 : 500.000Jarak peta = 24 cmJarak sebenarnya

Js = Jp/Skala

Js = 24 cm/1 : 500.000

Js = 24 cm × 500.000/1

Js = 12.000.000 cm/1

Js = 12.000.000 cm

Js = (12.000.000 : 100.000) km

Js = 120 km

'비상'

Skala dan Perbandingan

..

[tex]\boxed{ \begin{array}{lr} \boxed{\large{\sf{ \frac{JP}{ \: \: JS \: \: | \: \: s \: \: } } }}\\ \\\sf{Keterangan \ \: :}\\ \begin{aligned} \sf{ JS } &= \sf{ Jarak~Sebenarnya ~~~~~~~~~~}\\ \sf{ JP } &= \sf{ Jarak~Pada~Peta }\\\sf{ s } &= \sf{ Skala }\end{aligned} \end{array}}\begin{aligned} &\ \to \boxed{ \sf{JP = JS × s}} \\ &\ \to \boxed{ \sf{JS = \frac{JP}{s}} \: \: \: \: \: \: }\\ &\ \to \boxed{\: \: \sf{ s \: \:= \frac{JP}{JS}} \: \: \: \: \: } \end{aligned}[/tex]

Penyelesaian Soal

[tex]\begin{aligned} JS &= \frac{24~cm}{1:500.000} \\&= 24~cm × 500.000 \\&= 12.000.000~cm \\&= \boxed{\bold{\underline{120~km}}} \end{aligned}[/tex]
