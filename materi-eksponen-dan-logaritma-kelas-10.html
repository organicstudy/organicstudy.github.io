logaritma

========

x log x = 1

log 10 = 1

x log a . x log b = x log (a + b)

x log a : x log b = x log (a - b)

x log n . ⁿlog y = x log y

x log a = 1/ a log x

1/aⁿ = a^-n

========

a). log 0,001

= log (0,1)³

= 3 (log 0,1)

= 3 (log 1/10)

= 3 (log 10^-1)

= 3 . (-1) (log 10)

= -3 .(1)

= –3

=========

b). ²log 1/32√2 --> 1/aⁿ = a^-n

= ²log (32√2)^-1

= ²log (32 × 2^½)^-1

= ²log 32^-1 + ²log 2^-½

= ²log (2⁵)^-1 + (-½)(²log 2)

= ²log 2^-5 - ½ (1)

= -5(²log 2) - ½

= -5 (1) - ½

= (-10 - 1)/2

= -11/2

= -5 ½

= -5,5

30. Materi Logaritma, Kelas 10Tolong Dijawab


Bab Logaritma
Matematika SMA Kelas X

^ = pangkat

1] b = a⁴
    ᵃlog b = 4

   ᵃlog b - ᵇlog a = 4 - 1/(ᵃlog b)
                          = 4 - 1/4
                          = 16/4 - 1/4
                          = 15/4
                          = 3 3/4

2] ᵃlog b = 4
    ᵃlog c = 2

   (ᵃlog (bc)⁴)^(1/2) = (4 . (ᵃlog b + ᵃlog c))^(1/2)
                              = √(4 . (4 + 2))
                              = √(4 . 6)
                              = √4 x √6
                              = 2√6
