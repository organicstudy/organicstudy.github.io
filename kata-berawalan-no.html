1. Carilah 5 kata benda yang bisa dihitung ( countable ) beserta kalimatnya yang diawali dengan huruf vokal !

==>

Kata Benda ( Countable )

Apple ( Apel )Orange ( Jeruk )University ( Universitas )Eagle ( Elang )Aloe Vera ( Lidah Buaya )

Kalimatnya :

• She has an Apple ( dia mempunyai sebuah apel )

• I have an Orange ( saya mempunyai sebuah jeruk )

• Look ! , this is an University ( lihat ! , ini adalah sebuah universitas )

• Look ! , there is an Eagle in the sky ( lihat ! , itu seekor elang di langit )

• She has an Aloe Vera plant ( dia punya sebuah pohon lidah buaya )

2. Carilah 5 kata yang tidak dapat dihitung ( uncountable ) beserta kalimatnya !

==>

Kata Benda ( Uncountable )

Cake ( Kue )Paper ( Kertas )Cofee ( Kopi )Bread ( Roti )Sugar ( Sugar )

Kalimatnya :

• This is a piece of Cake ( ini adalah sepotong kue )

• That is a sheet of Paper ( itu adalah sehelai kertas )

• That is a cup of Cofee on the table ( itu adalah secangkir kopi diatas meja )

• This is a loaf of Bread ( ini adalah sepotong roti )

• That is a lump of Sugar in a bottle ( itu adalah gumpalan gula di dalam botol )

Penjelasan:

A = Kata keterangan benda yang diawali dengan huruf Konsonan

Contoh : a table, a lamp, a sugar.

An = Kata keterangan benda yang diawali dengan huruf Vokal

Contoh : an apple, an orange.
