Titik sudut berjumlah 8 titik (A, B, C, D, E, F, G, H).

Diagonal bidang yang sama panjang sebanyak 6 buah (AC = BD = EG = FH = AF = BE = CH = DG = AH = DE = BG = CF).

Diagonal ruang yang sama panjang sebanyak 4 buah (AG = BH = CE = DF).

10. Rumus volume dan luas permukaan bangun ruang sisi datar


Rumus-rumus bangun ruang sisi datar, langsung saja disimak pembahasan dibawah ini.

Pembahasan

Kubus adalah bangun ruang yang semua rusuknya sama panjang, dan semua sisinya sama luas dan bentuknya persegi.

Rumus-rumus pada kubus

Volume = rusuk x rusuk x rusuk

Jika rusuk = s, dan Volume = V

Maka:

V = s x s x s

s = ∛V

Luas permukaan kubus = 6s²

Luas alas kubus = s x s

s = √Luas

Materi tentang kubus dapat disimak juga di brainly.co.id/tugas/3661697

Balok adalah bangun ruang yang memiliki 12 rusuk dengan 3 ukuran, ukuran panjang, ukuran lebar dan ukuran tinggi.

Rumus-rumus pada balok

Volume balok = Luas alas x tinggi

Volume balok = p x l x t

p = Volume : (l x t)

l = Volume : (p x t)

t = Volume : (p x l)

Luas permukaan balok = 2 x (pl + pt + lt)

p = panjang

l = lebar

t = tinggi

Panjang semua rusuknya = 4 x (p + l + t)

Materi tentang Balok dapat disimak juga di brainly.co.id/tugas/11787450

Limas adalah bangun ruang yang alasnya berbentuk segi banyak, dan sisi tegaknya berbentuk segitiga.

Banyak sisi pada limas segi-n dapat dirumuskan sebagai n + 1.

Banyak rusuk pada limas segi-n dapat dirumuskan sebagai 2n.

Banyak titik sudut pada limas segi-n dapat dirimuskan sebagai n + 1.

Pada limas, namanya berdasarkan alasnya, jika alasnya segitiga, maka nama limas tersebut adalah limas segitiga, begitu seterusnya.

Rumus Volume Limas

= [tex]\frac{1}{3}[/tex] x luas alas x tinggi

Rumus luas permukaan Limas

= Luas alas + jumlah luas sisi tegak

Contoh soal tentang Limas dapat disimak juga di  brainly.co.id/tugas/1735712

Bangun prisma adalah bangun ruang sisi datar yang memiliki alas dan tutup yang sama bentuknya.

Banyak sisi pada prisma segi-n dapat dirumuskan sebagai n + 2.

Banyak rusuk pada prisma segi-n dapat dirumuskan sebagai 3n.

Banyak titik sudut pada prisma segi-n dapat dirimuskan sebagai 2n.

Rumus umum volume prisma ↓

[tex]\boxed{~Volume~=~Luas~alas~.~Tinggi~}[/tex]

Rumus umum Luas permukaan prisma ↓

[tex]\boxed{~Luas~Permukaan~prisma~=~2~.~Luas~alas~+~Keliling~alas~.~Tinggi~}[/tex]

30. tuliskan macam macam bangun ruang sisi datar berserta rumus luas dan volumenya​


Jawaban:

1. Kubus

Luas permukaan kubus = 6 x (luas persegi)

Volume = sisi x sisi x sisi = s³

2. Balok

Luas permukaan = (4 x (panjang x lebar)) + (2 x (lebar x tinggi))

Volume = panjang x lebar x tinggi

3. Tabung

Luas permukaan = 2 x (luas lingkaran) + luas selimut tabung

= 2 x phi x r² + 2r x tinggi

Volume = luas alas x tinggi

4. Prisma

Luas permukaan = luas alas + luas tutup + luas selimut prisma

Volume = luas alas (segitiga) x tinggi

5. Limas (segiempat dengan alas persegi)

Luas permukaan = luas alas (persegi) + 4xluas bidang tegak (segitiga)

= s² + 4(½ x alas x tinggi segitiga)

Volume = ⅓ x luas alas (persegi) x tinggi limas
