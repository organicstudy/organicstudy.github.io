5. Q.Banyak Susunan Kata,AnehBinAjaib....​


بِسْــــــــــــــــــمِ اللهِ الرَّحْمَنِ الرَّحِيْمِ

1.) aneh => 24 sususan kata

2.) Bin => 6 sususan kata

3.) ajaib => 60 sususan kata

Pendahuluan

Kaidah pencacahan ( counting rules ) merupakan sebuah cara atau aturan untuk menghitung seluruh kemungkinan yang bisa terjadi dalam suatu percobaan tertentu.

•

•

metode metode kaidah pencacahan

Aturan pengisian tempatPermutasikombinasi

• Permutasi

Permutasi adalah susunan yang berurutan dari semua atau sebagian elemen dari suatu himpunan.

Rumus:

[tex]\sf { \pink { { P \frac{n}{r} = \frac{n!}{( \: n - r \: )!} }}}[/tex]

Jika tdk ada huruf ganda, Rumus nya berlaku:

[tex]\sf { \pink { { P \frac{n}{n} = n! }}}[/tex]

• Kombinasi

Kombinasi merupakan suatu pengelompokan dari sebagian atau seluruh elemen dari suatu himpunan tanpa memperhatikan urutan susunan pemilihannya

Rumus:

[tex]\sf { \pink { { C \frac{n}{r} = \frac{n}{( \: n \: - \: r)r} }}}[/tex]

Pembahasan

hii sahabat brainly, kali ini kita akan membahas tentang kaidah pencacahan dgn metode Permutasi

-> yuk simak pembahasan berikut ini:

-------------------------

1.) [tex]\sf { \red { { aneh }}}[/tex]

• A = 1

• n = 1

• e = 1

• h = 1

→ Jumlah huruf : 4

→ huruf ganda : -

[tex]\sf { \pink { { P \frac{n}{n} = n! }}}[/tex] = [tex]P \frac{4}{4} = 4![/tex] = 4! = 4 × 3 × 2 × 1 = [tex] { \underline{ \boxed{ \blue{24 \: sususan \: kata}}}}[/tex]

----------::-----------::------------::------------

2.) [tex]\sf { \red { {bin }}}[/tex]

• b = 1

• i = 1

• n = 1

→ Jumlah huruf : 3

→ huruf ganda : -

[tex]\sf { \pink { { P \frac{n}{n} = n! }}}[/tex] = [tex]P \frac{3}{3} [/tex] = 3! = 3 × 2 × 1 =[tex] { \underline{ \boxed{ \blue{6 \: sususan \: kata}}}}[/tex]

----------::-----------::------------::------------

3.) [tex]\sf { \red { { ajaib }}}[/tex]

• A = 2

• j = 1

• i = 1

• b = 1

→ Jumlah huruf : 5

→ Huruf ganda : a ( 2 )

[tex]\sf { \pink { { p \frac{n}{r} = \frac{n!}{( \: n - r \: )} }}}[/tex] = [tex]p \frac{5}{2} = \frac{5!}{\: 2!} [/tex] = [tex] \frac{5 \times 4 \times 3 × 2 × 1}{2 × 1} [/tex] = [tex] { \underline{ \boxed{ \blue{60 \: sususan \: kata}}}}[/tex]

16. THUIZZ!!Banyak susunan kata "bayi ajaib"-,​


bayi ajaib

b = 2

a = 3

y = 1

i = 2

j = 1

Total Unsur = 9

Total Ganda = 2, 3, 2

Banyak Susunan :

= 9! / 2! × 3! × 2!

= 362.880 / 24

= 15.120 Susunan kata

Jawaban:

15.120 susunan

Penyelesaian

"bayi ajaib"

B = 2

A = 3

Y = 1

I = 2

J = 1

total huruf = 9huruf ganda = B(2), A(3), I(2)

Banyak susunan:

P = total huruf! / huruf ganda!

P = 9! / 2! × 3!

P = (9 × 8 × 7 × 6 × 5 × 4 × 3 × 2 × 1) / (2 × 1 × 3 × 2 × 1 × 2 × 1)

P = 362.880 / 24

P = 15.120 susunan.
