11. tegangan yang terukur pada resistor adalah...volt.​


Tegangan yang terukur pada resistor adalah A . 16 Volt

PEMBAHASAN

Pada suatu rangkaian seri yang terdiri dari Resistor , Induktor dan Kapasitor , maka bisa di cari impedansi rangkaian ini dengan cara :

[tex]Z = \sqrt{R^2+(X_L-X_C)^2}[/tex]

dimana :

Z = impedansi ( Ω )

R = resistansi ( Ω )

Xʟ = reaktansi induktif ( Ω )

Xc = reakstansi kapasitif ( Ω )

Frekuensi Resonansi yang dimiliki oleh rangkaian RLC bisa di cari dengan rumus :

[tex]\large \boxed{f=\frac{1}{2\pi \sqrt{LC} } }[/tex]

dimana :

f = frekuensi resonansi ( Hz )

L = induktansi ( Henry )

C = kapasitansi ( Farad )

Mari kita mencoba menyelesaikan soalnya

Diketahui:

V = 20 V

R = 8 Ω

Xʟ = 6 Ω

Ditanyakan:

Vʀ = ?

Penyelesaian:

Z² = R² + Xʟ²

Z² = 8² + 6²

Z² = 100

Z = 10 Ω

I = V / Z

I = 20 / 10

I = 2 A

Vʀ = I . R

Vʀ = 2 . 8

Vʀ = 16 Volt

27. cara mengukur resistor menggunakan ohm meter?​


                                                                                                                                                                                                                                                                                                                                                                                Besaran merupakan suatu pengukuran dalam fisika dan satuan adalah unit yang terukurnya .

I.Besaran : Sesuatu yang dapat diukur, memiliki nilai dan satuan  

Pembagian Besaran  

1.Berdasar Satuan  

a.Besaran Pokok : Besaran yang satuan sudah terdefinisikan dan berdiri sendiri  

Contoh :  

( J I Wa Ku S M P )  

Jumlah zat ( mol ), Intensitas cahaya ( Cd ), Waktu ( s ), Kuat arus ( A ), Suhu ( K ), Massa ( Kg ) dan Panjang ( m )  

b.Besaran Turunan : Besaran yang satuannya diturunkan dari satuan besaran pokok  

Contoh :  

Luas ( m2 ), Volume ( m3 ), Massa jenis ( Kg/m3 ), Kecepatan ( m/s ), Percepatan ( m/s2 ), Gaya ( N = kgm/s2 ), Energi ( J = Nm = Kgm2/s2 ), Daya ( Watt = J/s = Kgm2/s3 ), Tekanan ( N/m2 = Kg/ms2 ), Momentum ( Kgm/s ), Impuls ( Kgm/s ), Usaha (J)  

2.Berdasar Arah  

a.Besaran Skalar : Besaran yang hanya memiliki nilai saja  

Contoh : Semua besaran pokok termasuk besaran skalar, Luas, Volume, Kelajuan, Perlajuan, Massa Jenis, Energi, Daya dsb  

b.Besaran Vektor : Besaran yang memiliki nilai dan arah  

Contoh : Perpindahan, Kecepatan, Percepatan, Gaya, Momentum dsb  

II.Dimensi  

Merupakan gambaran bagaimana besaran tersebut tersusun atas besaran-besaran pokok.  

Fungsi dimensi : untuk menguji benar / setara tidaknya suatu persamaan ( Persamaan yang benar secara dimensi belum tentu benar secara fisis )  

Dimensi dari besaran – besaran pokok :  

Jumlah zat [ N ], Intensitas cahaya [ J ], Waktu [ T ], Kuat arus [ I ], Suhu [ θ ], Massa [ M ] dan Panjang [ L ]

Pembahasan  

Pertama nyalakan OHm meter, pilih nilai rentang pengukuran yang tepat. kemudian hubungkan dengan resistor seperti gambar. setelah itu baca nilainya

30. resistor terukur 120,5 ohm. jika ujung ujung resistor diberi beda potensial 1,5v maka berapa kuat arus yang lewat?​


Jawaban:

Kuat arus yang lewat = [tex]1,24 \times 10^{-2} A[/tex].

Penjelasan:

Diketahui :

R = 120,5 Ω

V = 1,5 V

Ditanya : I = ... ?

Jawab :

Kita menggunakan rumus :

[tex]I = \frac{V}{R}[/tex] dimana :

I = Kuat arus (satuannya Ampere dengan simbol A)

V= beda potensial  (satuannya Volt dengan simbol V)

R = Hambatan ( satuannya ohm dengan simbol Ω

[tex]I = \frac{1,5}{120,5}\\ I = 1,24 \times 10^{-2} A[/tex]
