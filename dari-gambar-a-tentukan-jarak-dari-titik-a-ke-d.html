2. Gambarlah titik A (−4, 2), B (−4, 9), C (2, 2), dan D (3, 9), pada koordinat Kartesius

Sistem koordinat kartesius merupakan sebuah sistem penentuan letak suatu titik/benda yang dinyatakan dengan koordinat (x, y).

Pembahasan

Untuk gambar titik koordinat soal no 1 dan 2 bisa dilihat pada lampiran

1. Titik koordinat A (1, −2), B (−3, 6), C (2, 8), dan D (−1, −5)

 a. Titik-titik yang berada pada kuadran I, II, III, dan IV.

Titik yang berada pada kuadran I adalah titik CTitik yang berada pada kuadran II adalah titik BTitik yang berada pada kuadran III adalah titik DTitik yang berada pada kuadran IV adalah titik A

 b. Jarak setiap titik dengan sumbu-x

Jarak titik A = 2 satuanJarak titik B = 6 satuanJarak titik C = 8 satuanJarak titik D = 5 satuan

 c. Jarak setiap titik dengan sumbu-y

Jarak titik A = 1 satuanJarak titik B = 3 satuanJarak titik C = 2 satuanJarak titik D = 1 satuan

2. Titik koordinat A (−4, 2), B (−4, 9), C (2, 2), dan D (3, 9)

 a. Jarak setiap titik dengan sumbu-x

Jarak titik A = 2 satuanJarak titik B = 9 satuanJarak titik C = 2 satuanJarak titik D = 9 satuan

 b. Jarak setiap titik dengan sumbu-y

Jarak titik A = 4 satuanJarak titik B = 4 satuanJarak titik C = 2 satuanJarak titik D = 3 satuan

 c. Jarak antara titik A dengan titik B adalah 7 satuan

 d. Jarak antara titik C (2, 2), dan D (3, 9)

    CD² = (x₂ - x₁)² + (y₂ - y₁)²

            = (3 - 2)² + (9 - 2)²

            = 1² + 7²

            = 1 + 49

            = 50

     CD = √50

     CD = 5√2

     CD = 7,07 satuan

  Jadi jarak titik C terhadap titik D adalah 7,07 satuan

Gambarlah titik A(-4, 2), B(-4, 9), C(2, 2) dan D(3, 9) pada koordinat kartesius

A. tentukan jarak setiap titik dengan sumbu -x

B. tentukan jarak setiap titik dengan sumbu -y

C. tentukan jarak antara titik A dengan titik B

D. tentukan jarak antara titik C dan D

Jawaban

Pendahuluan

Misal koordinat suatu titik P(x₁, y₁) dan Q(x₂, y₂)

Jarak titik P terhadap sumbu x = |y₁| ⇒ nilainya selalu positif

Jarak titik P terhadap sumbu y = |x₁| ⇒ nilainya selalu positif

Jarak titik P dan Q = √((x₁ - x₂)² + (y₁ - y₂)²)

Pembahasan

A) Jarak titik A(-4, 2) terhadap sumbu x = |2| = 2 satuan

Jarak titik B(-4, 9) terhadap sumbu x = |9| = 9 satuan

Jarak titik C(2, 2) terhadap sumbu x = |2| = 2 satuan

Jarak titik D(3, 9) terhadap sumbu x = |9| = 9 satuan

B) Jarak titik A(-4, 2) terhadap sumbu y = |-4| = 4 satuan

Jarak titik B(-4, 9) terhadap sumbu y = |-4| = 4 satuan

Jarak titik C(2, 2) terhadap sumbu y = |2| = 2 satuan

Jarak titik D(3, 9) terhadap sumbu y = |3| = 3 satuan

C) Jarak titik A(-4, 2) dan B(-4, 9)

= √((-4 - (-4))² + (2 - 9)²)

= √((0)² + (-7)²)

= √(0 + 49)

= √49

= 7 satuan

D) Jarak titik C(2, 2) dan D(3, 9)

= √((2 - 3)² + (2 - 9)²)

= √((-1)² + (-7)²)

= √(1 + 49)

= √(50)

= √(25 × 2)

= √25 × √2

= 5 × √2

= 5√2 satuan

Kesimpulan

Jarak titik (x, y) terhadap sumbu x = |y| ⇒ nilainya selalu positif

Jarak titik (x, y) terhadap sumbu y = |x| ⇒ nilainya selalu positif

Jarak titik (x₁, y₁) dan (x₂, y₂) = √((x₁ - x₂)² + (y₁ - y₂)²)

18. Gambarlah titik A (-4,2), B (-4,9), C (2,2), dan D (3,9), pada koordinat Kartesius. a.Tentukan jarak setiap titik dengan sumbu-X.b. Tentukan jarak setiap titik dengan sumbu-Y.c. Tentukan jarak antara titik A dan titik B.d. Tentukan jarak antara titik C dan titik D. Tolong di jawab yaa ​


Jawaban:

Pendahuluan

Misal koordinat suatu titik P(x₁, y₁) dan Q(x₂, y₂)

Jarak titik P terhadap sumbu x = |y₁| ⇒ nilainya selalu positif

Jarak titik P terhadap sumbu y = |x₁| ⇒ nilainya selalu positif

Jarak titik P dan Q = √((x₁ - x₂)² + (y₁ - y₂)²)

Pembahasan

A) Jarak titik A(-4, 2) terhadap sumbu x = |2| = 2 satuan

Jarak titik B(-4, 9) terhadap sumbu x = |9| = 9 satuan

Jarak titik C(2, 2) terhadap sumbu x = |2| = 2 satuan

Jarak titik D(3, 9) terhadap sumbu x = |9| = 9 satuan

B) Jarak titik A(-4, 2) terhadap sumbu y = |-4| = 4 satuan

Jarak titik B(-4, 9) terhadap sumbu y = |-4| = 4 satuan

Jarak titik C(2, 2) terhadap sumbu y = |2| = 2 satuan

Jarak titik D(3, 9) terhadap sumbu y = |3| = 3 satuan

C) Jarak titik A(-4, 2) dan B(-4, 9)

= √((-4 - (-4))² + (2 - 9)²)

= √((0)² + (-7)²)

= √(0 + 49)

= √49

= 7 satuan

D) Jarak titik C(2, 2) dan D(3, 9)

= √((2 - 3)² + (2 - 9)²)

= √((-1)² + (-7)²)

= √(1 + 49)

= √(50)

= √(25 × 2)

= √25 × √2

= 5 × √2

= 5√2 satuan

Kesimpulan

Jarak titik (x, y) terhadap sumbu x = |y| ⇒ nilainya selalu positif

Jarak titik (x, y) terhadap sumbu y = |x| ⇒ nilainya selalu positif

Jarak titik (x₁, y₁) dan (x₂, y₂) = √((x₁ - x₂)² + (y₁ - y₂)²)
