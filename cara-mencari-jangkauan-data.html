1). Statistika adalah sebuah ilmu yang mempelajari bagaimana cara merencanakan, mengumpulkan, menganalisis, lalu menginterpretasikan, dan akhirnya mempresentasikan data. Singkatnya, statistika adalah ilmu yang bersangkutan dengan suatu data. Istilah "Statistika" berbeda dengan "Statistik".

2). Nilai rata-rata – pada suatu kelompok yaitu jumlah nilai data kemudian dibagi dengan banyaknya data tersebut, biasanya nilai rata-rata digunakan untuk mengevaluasi data sehingga lebih cepat dan menggambarkan seluruh data serta tidak dapat digunakan untuk menentukan nilai data tertentu di antara sekelompok data, sebagai contoh misalnya nilai rata-rata budi yaitu 85 jadi tidak bisa disimpulkan bahwa nilai untuk pelajaran Matematika Budi lebih dari 75.

Nilai rata-rata tersebut bisa dipakai untuk membandingkan kelompok data yang satu dengan yang lainnya, seperti yg nilai rata-rata kelas yaitu 85 sedangkan nilai rata-rata kelas B yaitu 87 itu berarti kelas B lebih tinggi daripada kelas A..

17. Carilah jangkauan Q1 Q2 Q3 dari data di bawah ini a. 4,8,6,7,5,7,8,9,9,8,13,8,10,12,11 b. 7,8,8,9,7,6,5,8


Jawab:

Penyelesaian dibawah

Penjelasan dengan langkah-langkah:

a. Data urut :

4,5,6,7,7,8,8,8,8,9,9,10,11,12,13

Xmaks = 13 dan Xmin = 4

Maka Jangkauan data = 13 - 4 yaitu 9

Banyak data = 15 data ( ganjil )

Keterangan data ke - ( x )

Q1 = Q2 - ½ } x8 - ½ } x4 yaitu 7

Q2 = xn + 1 / 2 } x15 + 1 / 2 } x16 / 2 } x8

Data ke 8 data tersebut adalah 8

Q3 = Q2 + ½ } x8 + ½ } x12 yaitu 10

b. Data urut :

5,6,7,7,8,8,8,9

Xmaks = 9 dan Xmin = 5

Maka Jangkauan data = 9 - 5 yaitu 4

Banyak data = 8 data ( genap )

Keterangan data ke - ( x )

Kuartil Bawah atau Q1

Q1 = Q2 - ( Q2 - 0,5 / 2 )

Q1 = x4,5 - ( x4,5 - 0,5 / 2 )

Q1 = x4,5 - x2 yaitu x2,5 adalah 6,5

Kuartil Tengah atau Q2

Q2 = xn/2 + xn/2+1 / 2

Q2 = x8/2 + x8/2+1 / 2

Q2 = x4 + x5 / 2 yaitu x4,5

Data ke 4,5 data tersebut adalah 7,5

Kuartil atas atau Q3

Q3 = Q2 + ( Q2 - 0,5 / 2 )

Q3 = x4,5 + ( x4,5 - 0,5 / 2 )

Q3 = x4,5 + x2 yaitu x6,5 adalah 8
