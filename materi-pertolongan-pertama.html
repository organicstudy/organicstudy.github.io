3. Tolong dong, baru pertama kali belajar materi ini​


Nilai dari [tex] {( {4}^{ - 1} + {3}^{ - 2} + {7}^{ - 1}) }^{- 1} [/tex] adalah C. 1,98

Pendahuluan

Bilangan berpangkat merupakan bilangan yang akan menghasilkan hasil dari perkalian berulang dengan syarat memiliki bilangan yang sama.

Jadi bilangan eksponen bisa disebut juga sebagai bentuk dari sebuah bilangan yang di kalikan yaitu dengan bilangan yang sama dan dilakukan berulang ulang, eksponen di sebut dengan pangkat yang menunjukan nilai dari derajat kepangkatan.

Bilangan berpangkat adalah perkalian angka tersebut dengan angka yang sama sebanyak pangkat

a³ = a x a x a

Akar pangkat 2 adalah hasil dari suatu pangkat pecahan yaitu 1/2 √a = a^½

Akar pangkat 3 adalah pangkat ⅓ dari suatu bilangan ³√a = a^⅓

Sifat Pada Bilangan Berpangkat :

Jika pembagian maka pangkat dikurang. [tex] a^m \div a^n = a^{(m-n)} [/tex]Jika dikali maka pangkat ditambah. [tex] a^m \times a^n = a^{(m+n)} [/tex][tex] (a^m)^n = a^{m\times n} [/tex][tex] (ab)^n =a^nb^n [/tex][tex] \sqrt[n]{ {a}^{m} } = {a}^{ \frac{m}{n} } [/tex][tex] (\frac{a}{b})^n = \frac{a^n}{b^n} [/tex][tex] a^{-n} = \frac{1}{a^n} [/tex][tex] a^0 = 1 [/tex]

Untuk mengerjakan soal di atas kita simak penjelasan di bawah ini :

PembahasanDiketahui:

Nilai dari [tex] {( {4}^{ - 1} + {3}^{ - 2} + {7}^{ - 1}) }^{- 1} [/tex] adalah

Ditanya:Hasilnya ?

Jawab:

[tex] = {( {4}^{ - 1} + {3}^{ - 2} + {7}^{ - 1}) }^{- 1} [/tex]

[tex] = { (\frac{1}{4} + \frac{1}{ {3}^{2} } + \frac{1}{7} )}^{ - 1} [/tex]

[tex] = { (\frac{1}{4} + \frac{1}{9 } + \frac{1}{7} )}^{ - 1} [/tex]

[tex] = {( \frac{63 + 28 + 36}{252} )}^{ - 1} [/tex]

[tex] = {( \frac{127}{252} )}^{1} [/tex]

[tex] = \frac{252}{127} [/tex]

[tex] = 1 \frac{125}{127} [/tex]

[tex] = 1.98[/tex]

Kesimpulan:

Nilai dari [tex] {( {4}^{ - 1} + {3}^{ - 2} + {7}^{ - 1}) }^{- 1} [/tex] adalah C. 1,98

15. turunan pertama fungsi f(x) = (3x-2) (5x-1)³ adalah f'(x). f'(1) =......... -tolong bantu yaa-materi turunan kelas 11


Jawab:

Penjelasan dengan langkah-langkah:

[tex]f(x)=(3x-2)(5x-1)^3[/tex]

Fungsi tersebut adalah bentuk fungsi perkalian f(x) dan g(x), maka sesuai rumus turunan hasil kali dua fungsi :

F(x) = f(x).g(x) ----> F'(x) = f'(x).g(x) + f(x).g'(x)

diperoleh :

[tex]F'(x)=3(5x-1)^3+3(3x-2).(5x-1)^2.5\\F'(x)=3(5x-1)^3+15(3x-2)(5x-1)^2[/tex]

Langsung substitusi saja untuk x = 1, sehingga :

[tex]F'(1)=3(5-1)^3+15(3-2)(5-1)^2=192+240=432[/tex]

Semoga membantu.

Penjelasan dengan langkah-langkah:

f(x) = (3x-2) (5x-1)³

a = 3x - 2

a' = 3

b = (5x - 1)³

b' = 3(5)(5x - 1)² = 15(5x - 1)²

f'(x) = a'b + ab'

= 3(5x -1)³ + 15(3x - 2)(5x - 1)²

f'(1) = 3(5.1 - 1)³ + 15(3.1 -2)(5.1 -1)²

= 3(4)³ + 15(1)(4²)

= 3(64) + 15(16)

= 192 + 240

= 432

30. dua dadu di lempar bersama-sama. peluang muncul mata dadu pertama 3 dan dadu kedua 5 adalah... tolong di bantu. saya minus dalam materi peluang


S=6X6 = 36

A = mata dadu pertama 3 = 1
P(A) = 1/36

B = mata dadu kedua 5 = 1
P(B) = 1/36

P(A U B) = P(A) + P(B)
= 1/36 + 1/36
= 2/36

Maaf, kalau salah.1/36 + 1/36 = 2/ 36

1/36 itu berasal dari => ( dalam 1 dadu hanya ada terdapat 1 angka 3 dan 1 angka 5 ) ( sedangkan 36 nya berasal dari jumlah kedua mata dadu )
maka di buat 1/36 + 1/36 dan hasilnya = 2/36
