1. Suatu senyawa organik (mr = 80) mengandung 60% karbon, 5% hidrogen, dan sisanya nitrogen. Rumus senyawa tersebut adalah ...


Rumus senyawa merupakan rumus molekul yang dikalikan dengan n. Rumus senyawa berdasarkan soal di atas adalah [tex]C_{4}H_{4}N_{2}[/tex]

Penjelasan dengan langkah-langkah:

Diketahui:

Mr senyawa = 80%Carbon = 60%%Hidrogen = 5%%Nitrogen = 100%-60%-5% = 35%Ar C = 12Ar H = 1Ar N = 14

Ditanya:

Rumus senyawa?

Jawab:

Langkah pertama yang harus kita lakukan adalah menghitung masing-masing mol.

Mol Carbon:

[tex]n C = \frac{60}{12} = 5[/tex]

Mol Hidrogen:

[tex]nH = \frac{5}{1}=5[/tex]

Mol Nitrogen:

[tex]nN = \frac{35}{14} = 2,5[/tex]

nC:nH:nN = 5:5:2,5

Lalu kita sederhanakan, sehingga:

nC:nH:nN = 2:2:1

Sehingga rumus molekulnya menjadi [tex]C_{2}H_{2}N[/tex]

Mr rumus senyawa = Mr rumus molekul.n

80 = (12.2+1.2+14)n

n = 2

Sehingga rumus senyawanya [tex]C_{4}H_{4}N_{2}[/tex]

16. Suatu senyawa organik (Mr=80) mengandung 60% karbon 5%hidrogen,dan sisanya nitrogen tentukan rumus empiris dan rumus molekul senyawa organik tersebut


Rumus empiris dan rumus molekul senyawa berturut-turut adalah (C₂H₂N)n dan C₄H₄N₂. Rumus molekul didapatkan dari rumus empiris dan data Mr-nya. rumus empiris dapat ditentukan dari perbandingan mol masing-masing unsur dalam senyawa. mol unsur dapat ditentukan dari massa unsur banding Ar-nya.

Pembahasan  

Pada soal di atas diketahui beberapa data yaitu:  

% C = 60 %  

% H = 5 %  

Ar C = 12, H = 1, N = 14  

Mr = 80 gram/ mol  

Adapun yang ditanyakan pada soal adalah rumus empiris rumus molekul dari senyawa tersebut.  

Langkah awal yang harus dilakukan untuk menentukan rumus molekul senyawa adalah mengetahui rumus empirisnya. Rumus empiris dapat ditentukan menggunakan massa masing-masing unsur tersebut dalam senyawa dibanding dengan Ar-nya. Massa unsur dalam senyawa dapat ditentukan dari massa zat yang dihasilkan. Berikut langkah yang dapat dilakukan dalam penentuan rumus molekul.  

Step-1 (menentukan massa unsur)  

massa unsur dapat ditentukan menggunakan data % massa unsur yang diketahui pada soal. Kalian dapat melakukan permisalan untuk massa senyawa. jika dimisalkan massa senyawa adalah 100 gram. Maka massa masing-masing-unsur penysusun senyawa dapat ditentukan sebagai berikutpenghitungannya:  

massa C = % C x 100 gram

             = 60% x 100 gram

             = 60 gram  

massa H = % H x 100 gram

              = 5 % x 100 gram

              = 5 gram

Massa N = senyawa – (massa C + massa H)  

              = 100 – (60 + 5)  

              = 100 – 65  

              = 35 gram  

Didapatkan massa C, H dan N berturut-turut adalah 60 gram, 5 gram dan 35 gram.

Step-2 (menentukan mol unsur penyusun)  

Mol unsur penyusun senyawa dapat ditentukan menggunakan data massa unsur yang telah didapatkan dibandingkan dengan massa atom relatifnya. kalian dpaat menggunakan massa unsur pada penghitungan sebelumnya dan data Ar yang telah diketahui pada soal. berikut mol masing-masing unsur penyusun senyawa organik tersebut  

mol C = Massa C / Ar C  

          = 60/ 12  

          = 5 mol  

mol H = massa H/ Ar H  

          = 5/ 1  

          = 5 mol  

mol N = massa N/ Ar N  

          = 35/ 14

          = 2,5 mol  

didapatkan mol C, H dan N berturut-turut adalah 5 mol, 5 mol, dan 2,5 mol

Step-3 (menentukan rumus empiris)  

Rumus empiris dapat ditentukan dari perbandingan mol masing-masing unsur penyusun senyawa tersebut. Berikut cara menentukan rumus empirisnya:  

mol C : mol H : mol N  

   5     :    5      :    2,5     (dibagi 2,5)  

   2     :    2      :     1  

Jadi rumus empirisnya dapat dituliskan sebagai (C₂H₂N) n.  

Final Step (menentukan rumus molekul)

Rumus molekul dapat ditentukan dengan menggunakan data Mr dan rumus empiris yang telah didapatkan. Berikut penghitungannya.  

(C₂H₂N) n = Mr  

((12 x 2) + (1 x 2) + (14 x )) n = 80  

(24 + 2 + 14) n = 80

40 n = 80

    n = 2

Rumus molekulnya dapat dituliskan menjadi:  

(C₂H₂N) n = (C₂H₂N) 2 = C₄H₄N₂  

jadi dapat disimpulkan bahwa rumus empiris dan rumus molekulnya berturut-turut adalah (C₂H₂N)n dan C₄H₄N₂.
