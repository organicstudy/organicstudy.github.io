Diketahui :

1 kilo jeruk + 3 kilo Apel = Rp. 90,000

2 kilo jeruk + 1 kilo Apel = Rp. 55,000

Ditanyakan :

Harga 3 kilo jeruk + 1 kilo Apel ?

Jawab :

Soal ini membahas tentang sistem persamaan linear dua variabel atau SPLDV Didalam SPLDV terdapat beberapa metode yang digunakan yaitu metode eliminasi subsitusi grafik determinan dll..

Kita akan gunakan metode Eliminasi untuk mengerjakan ini , maka ;

Model matematika soal diatas adalah :

misal :

x = jeruk

y = Apel

maka,

x + 3y = 90,000

2x + y = 55,000

Metode Eliminasi :

eliminasi x untuk dapat kan nilai y maka :

x + 3y = 90,000 |×2

2x + y = 55,000 |×1

2x + 6y = 180,000

2x + y = 55,000

-------------------------- (-)

5y = 125,000

y = 125,000 : 5

y = 25,000

substitusi kan y = 25,000 ke :

2x + y = 55,000

2x + (25,000) = 55,000

2x = 55,000 - 25,000

2x = 30,000

x = 30,000 : 2

x = 15,000

jadi, harga

x = jeruk 1 kilonya Rp. 15,000

y = Apel 1 kilonya Rp. 25,000

maka, total harga dari :

= 3 kilo jeruk + 1 kilo Apel

= 3x + y

= 3(15,000) + (25,000)

= 45,000 + 25,000

= Rp. 70,000

15. 2 kilo dan 1 kilo apel dengan harga Rp15.000 sedangkan Ayah membeli 1 kilo mangga dan 2 kilo apel seharga Rp18.000 Berapakah harga 5 kilo mangga dan 3 kilo apel


materi: Sistem persamaan linear dua variabel
kelas: 2 SMP

Diketahui:
2 kilo mangga + 1 kilo apel = 15.000
1 kilo mangga + 2 kilo apel = 18.000

Ditanya:
5 kg mangga + 3 kg apel

Jawab:

2 kilo mangga + 1 kilo apel = 15.000
1 kilo mangga = 18.000 - 2 kg apel

maka,

2 (18.000 - 2 kilo apel) + 1 kilo apel = 15.000
36.000 - 4 kilo apel + 1 kilo apel = 15.000
- 4 kilo apel + 1 kilo apel = 15.000 - 36.000
- 3 kilo apel = - 21.000
1 kilo apel = - 21.000 / 3
1 kilo apel = 7.000

2 kilo mangga + 7.000 = 15.000
2 kilo mangga = 15.000 - 7.000
2 kilo mangga = 8.000
1 kilo mangga = 8.000 / 2
1 kilo mangga = 4.000

jadi,
5 kilo mangga + 3 kilo apel =
5 (4.000) + 3 (7.000)
20.000 + 21.000
41.000

jadi, harga 5 kilo mangga dan 3 kilo apel adalah 41.000

17. harga 2 kilo jeruk dan 3 kilo apel rp194.000 harga 5 kilo jeruk dan 2 kilo apel rp221.000 maka harga 3 kilo jeruk dan 2 kilo apel =​


Harga 3 kilo jeruk dan 2 kilo apel adalah Rp171.000,00.

Pembahasan:

Sistem Persamaan Linear Dua Variabel (SPLDV) adalah persamaan matematika yang terdiri dari dua variabel dan semua penyusunnya berpangkat satu.

Cara-cara menyelesaikan persamaan linear dua variabel:

Grafik. Prinsipnya, kedua persamaan dijadikan persamaan garis lurus, dan titik potong kedua garis merupakan penyelesaiannya.Substitusi. Prinsipnya mengubah nilai variabel lain ke dalam satu bentuk variabel saja.Eliminasi. Prinsipnya "menghilangkan" variabel lain hingga tersisa satu variabel dan konstanta saja.Gabungan. Pertama-tama gunakan eliminasi untuk mendapat salah satu nilai variabel, setelah itu substitusikan ke salah satu persamaan untuk mendapat nilai variabel lainnya.

-

Diketahui:

Harga 2 kilo jeruk dan 3 kilo apel adalah Rp194.000,00.

Harga 5 kilo jeruk dan 2 kilo apel adalah Rp221.000,00.

-

Ditanya:

Harga 3 kilo jeruk dan 2 kilo apel adalah...

-

Penyelesaian:

Misalkan harga satu kilo jeruk = x dan harga satu kilo apel = y.

[tex]2x + 3y = 194.000[/tex] ... persamaan 1

[tex]5x + 2y = 221.000[/tex] ...persamaan 2

Buat koefisien y kedua persamaan menjadi sama dengan cara mengalikan persamaan 1 dengan 2 dan mengalikan persamaan 2 dengan 3. Setelah itu, eliminasikan.

[tex]4x + 6y = 388.000[/tex]

[tex]15x + 6y = 663.000[/tex]

_____________________ -

[tex]-11x = -275.000[/tex]

[tex]\boxed{x = 25.000}[/tex]

.

Sekarang, substitusikan [tex]x = 25.000[/tex] ke salah satu persamaan. Kali ini akan disubstitusikan ke persamaan 2.

[tex]5x + 2y = 221.000[/tex]

[tex]5(25.000) + 2y = 221.000[/tex]

[tex]125.000 + 2y = 221.000[/tex]

[tex]2y = 96.000[/tex]

[tex]\boxed{y = 48.000}[/tex]

.

Harga satu kilo jeruk adalah Rp25.000,00 dan harga satu kilo apel adalah Rp48.000,00.

Maka harga 3 kilo jeruk dan 2 kilo apel:

[tex]3(Rp25.000,00) + 2(Rp48.000,00)[/tex]

[tex]= Rp75.000,00 + Rp96.000,00[/tex]

[tex]\boxed{\boxed{= Rp171.000,00}}[/tex]

-

Kesimpulan:

Jadi, harga 3 kilo jeruk dan 2 kilo apel adalah Rp171.000,00.
