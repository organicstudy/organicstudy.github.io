16. hasil panen kebun doni 90kg, hasil panen kebun Eko 10kg lebih sedikit dari hasil panen di kebun Doni, sedangkan hasil panen di kebun Ali 2 kali lebih banyak dari hasil panen di kebun Doni. berapakah jumlah hasil panen di kebun mereka? ​


MATEMATIKA

SISTEM PERSAMAAN TIGA VARIABEL

Misalkan :

Doni = D

Eko = E

Ali = A

Dari soal tersebut didapat persamaan sebagai berikut :

(1) D = 90kg

(2) E = D - 10kg

(3) A = 2D

E = 90kg - 10kg = 80kg

A = 2•(90kg) = 180kg

Jumlah hasil panen di kebun mereka :

= D + E + A

= 90kg + 80kg + 180kg

= 350kg

= 35%

Padi x 15 = 35 x Ketela

Padi x 15 = 35 x 45

Padi = 35 x 3 = 105 ton

(gandum - kacang) = 20 x 45/15

(gandum - kacang) = 20 x 3 = 60 ton

Panen seluruhnya/ketela = 100%/15%

Panen seluruhnya x 15 = 100 x ketela

Panen seluruhnya = 100 x 45/15
