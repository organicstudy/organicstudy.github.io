Persamaan reaksi

(2) CH₃CH = CHCH₃ + HCl ---> CH₃CH₂CHClCH₃

Alkena (ikatan rangkap 2 (=) ) dapat mengalami reaksi adisi (pemutusan ikatan rangkap) dengan asam halida HX (HF, HCl, HBr, HI) . Reaksi adisi dengan asam halida menggunakan aturan Markovnikov yaitu yang kaya makin kaya dan yang miskin makin miskin, maksudnya adalah atom H dari asam halida (HX) akan berikatan dengan C ikatan rangkap yang memiliki atom H lebih banyak, sehingga atom H nya akan semakin banyak (kaya).

Contoh :

CH₂ = CH - CH₃ + HCl ---> CH₃ - CHCl - CH₃  

Atom H pada asam halida HCl berikatan dengan atom C yang berikatan rangkap yaitu CH₂ karena jumlah atom H lebih banyak dari CH. Oleh karena itu atom Cl pada asam halida HCl berikatan dengan atom C yang berikatan rangkap yaitu CH.

20. Pada reaksi ZnS+O2->ZnO+SO2 setelah persamaan reaksi tersebut disetarakan,maka koefisien reaksi dari kiri kekanan berturut-turut adalah


ZnS + O2 ==>ZnO + SO2 (belum setara)
2ZnS + 3O2 ==>2ZnO + 2SO2 (sudah setara)
koefisien reaksi berturut turut yaitu
2 3 2 2Persamaan Reaksi
Kimia X

aZnS + bO₂ → cZnO + dSO₂

Zn ⇒ a = c
S ⇒ a = d
O ⇒ 2b = c + 2d

Misalkan a = 2, maka c = 2, dan d = 2
Substitusikan a, c, dan d ke 2b = c + 2d

2b = 2 + 2(2)
b = 3

Jadi a : b : c : d = 2 : 3 : 2 : 2

Sehingga 2ZnS + 3O₂ → 2ZnO + 2SO₂
