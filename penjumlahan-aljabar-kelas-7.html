28. 2. Tentukanlah jumlah dari : a. 5x + 3y dan 2x -3y b. 4a + b - 3c - d dan 2a + 3b - 2c + 3d c. 3a + 5b + 3c + 6d dan 6a - 7b - 7c - 2d d. 5p + 2r - 3s + 6t dan 7p - 2r - 3s - 6t e. 4x - 2y - 4z - 8 dan 4x + 2y + 5z + 8 MOHON DIJAWAB :D(Aljabar kelas 7 bab 2)


a. 5x+3y + (2x-3y) = 7x
b. 4a+b-3c-d + (2a+3b-2c+3d) = 6a+4b-c+2d
c. 3a+5b+3c+6d + (6a-7b-7c-2d) = 9a-2b-4c+4d
d. 5p+2r-3s+6t + (7p-2r-3s-6t) = 12p-6s
e. 4x-2y-4z-8 + (4x+2y+5z+8) = 8x+z
[tex]jawaban :[/tex]

[tex]a.)(5x + 3y) + (2x - 3y) \\ = 5x + 3y + 2x - 3y\\ = 5x + 2x + 3y - 3y \\ = 7x [/tex]

[tex]b.)(4a + b - 3c - d) + (2a + 3b - 2c + 3d) \\ = 4a + b - 3c - d + 2a + 3b - 2c + 3d \\ = 4a + 2a + b + 3b - 3c - 2c - d + 3d \\ = 6a + 4b - 5c + 2d[/tex]

[tex]c.)(3a + 5b + 3c + 6d) + (6a - 7b - 7c - 2d) \\ = 3a + 5b + 3c + 6d + 6a - 7b - 7c - 2d \\ = 3a + 6a + 5b - 7b + 3c - 7c + 6d - 2d \\ = 9a - 2b - 4c + 4d[/tex]

[tex]d.)(5p + 2r - 3s + 6t) + (7p - 2r - 3s - 6t) \\ = 5p + 2r - 3s + 6t + 7p - 2r - 3s - 6t \\ = 5p + 7p + 2r - 2r - 3s - 3s + 6t - 6t \\ = 12p - 6s[/tex]

[tex]e.)(4x - 2y - 4z - 8) + (4x + 2y + 5z + 8) \\ = 4x - 2y - 4z - 8 + 4x + 2y + 5z + 8 \\ = 4x + 4x - 2y + 2y - 4z + 5z - 8 + 8 \\ = 8x + z [/tex]
