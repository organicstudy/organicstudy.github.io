8. Buatlh contoh pengurangan dan penjumlahan aljabar dalam kehidupan sehari-hari​


Jawaban:

Anna pergi ke toko buku. Kemudian Anna membeli 3 buku dan 5 pensil dengan harga Rp.11.000,00. Jika Anna membeli lagi 1 buku dan 2 pensil untuk adiknya dengan harga Rp.4.000,00, maka berapakah harga 1 pensil dan 1 buku????

Jawab :

Misalkan :

B = Buku

P = Pensil

Maka :

3B + 5P = Rp. 11.000,00

B + 2P = Rp. 4.000,00

B = ???

P = ??

Maka untuk mencari B dan P kita harus ubah B + 2P = Rp. 4.000,00 menjadi B saja, maka :

B + 2P = Rp. 4.000,00

B + 2P - 2P = Rp. 4.000,00 - 2P

B = Rp. 4.000,00 - 2P

Jika B = Rp. 4.000,00 - 2P, maka 3B + 5P = Rp. 11.000,00, menjadi :

3( Rp. 4.000,00 - 2P) + 5P = Rp. 11.000,00

Rp. 12.000,00 - 6P + 5P = Rp. 11.000,00

Rp. 12.000,00 - P = Rp. 11.000,00

Rp. 12.000,00 - P - Rp.12.000,00 = Rp. 11.000,00 - Rp.12.000,00

-p = -Rp.1.000

-p x -1 = -Rp.1.000 x -1

p = Rp.1.000,00

Jadi harga 1 pensil sama dengan Rp.1.000,00

Contoh soal penjumlahan / pengurangan bentuk aljabar nomor 1

Selesaikan operasi hitung penjumlahan dan pengurangan berikut ini.

1. 5x + 2y – 19x + 10y

2. -5x + z – 4y + 4x + 2z – y

3. -2x + 5y – 7z + -2x – 3y – 3z

4. 5ab + 2ac – 7c + 2ab – 4ac + 3c

Pembahasan

Cara menjawab penjumlahan / pengurangan bentuk aljabar yaitu dengan menjumlahkan / mengurangkan koefisien yang variabelnya sama. Contohnya x dengan x, y dengan y, xy dengan xy, ab dengan ab, x2 dengan x2 dan lainnya.

1. 5x + 2y – 19x + 10y = (5x – 19x) + (2y + 10y) = -14x + 12y

2. -5x + z – 4y + 4x + 2z – y = (-5x + 4x) – (4y + y) + (z + 2z) = -x – 5y + 3z

3. -2x + 5y – 7z + -2x – 3y – 3z = (-2x – 2x) + (5y – 3y) – (7z + 3z) = -4x + 2y – 10z

4. 5ab + 2ac – 7c + 2ab – 4ac + 3c = (5ab + 2ab) + (2ac – 4ac) + (-7c + 3c) = 7ab – 2ac – 4c

26. 5 contoh soal penjumlahan pada aljabar fungsi​


Penjelasan dengan langkah-langkah:

Jika f(x) = 1 dan g(x) = 2x2 + 3 maka f(x) + g(x) = …

A. 2x2 + 4

B. 3x2 + 3

C. 4x2 + 3

D. 2x2 + 2

E. 2x2 + 5

Pembahasan

f(x) + g(x) = 1 + 2x2 + 3

f(x) + g(x) = 2x2 + 4

Soal ini jawabannya A.

Jika f(x) =

\frac {x} {x - 1} dan g(x) =

\frac {x} {x + 1} maka f(x) + g(x) = …

A.

\frac {2x^2} {x^{2} + 1}

B.

\frac {2x^2} {x^{2} - 1}

C.

\frac {2x^{2} -2} {x^{2} - 1}

D.

\frac {2} {x^{2} + 1}

E.

\frac {2x} {x^{2} - 1}

Pembahasan

f(x) + g(x) =

\frac {x} {x - 1} +

\frac {x} {x + 1}

f(x) + g(x) =

\frac {x (x + 1) + x (x - 1)} {(x - 1) (x + 1)}

f(x) + g(x) =

\frac {x^{2} + x + x^{2} - x} {x^{2} - 1}

f(x) + g(x) =

\frac {2x^2} {x^{2} - 1}

Koefisien

Konstanta

Nilai tetap yang tidak berubah

Suku

Nilai yang tidak memuat dengan variable
