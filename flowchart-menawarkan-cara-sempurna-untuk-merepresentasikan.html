4. Buatlah algoritma menggunakan representasi uraian deskripsi, flowchart dan pseudocode untuk permasalahan berikut ini:a. Menghitung luas dan keliling lingkaran b. Menghitung luas segitiga​


Jawaban:

Jawaban pada penjelasan

Penjelasan:

A. Berikut adalah representasi uraian deskripsi, flowchart, dan pseudocode untuk menyelesaikan permasalahan menghitung luas dan keliling lingkaran:

Uraian Deskripsi:

Diberikan sebuah lingkaran dengan jari-jari r.Tujuan dari algoritma ini adalah menghitung luas dan keliling lingkaran tersebut.

Flowchart:

Mulai

   Masukkan jari-jari r

   Hitung luas dengan rumus luas = πr^2

   Hitung keliling dengan rumus keliling = 2πr

   Tampilkan hasil luas dan keliling

Selesai

Pseudocode:

START

INPUT r

luas = πr^2

keliling = 2πr

PRINT luas, keliling

STOP

B. Berikut adalah representasi uraian deskripsi, flowchart, dan pseudocode untuk menyelesaikan permasalahan menghitung luas segitiga:

Uraian Deskripsi:

Diberikan sebuah segitiga dengan alas a dan tinggi t.Tujuan dari algoritma ini adalah menghitung luas segitiga tersebut.

Flowchart:

Mulai

   Masukkan alas a dan tinggi t

   Hitung luas dengan rumus luas = (a * t) / 2

   Tampilkan hasil luas

Selesai

Pseudocode:

START

INPUT a, t

luas = (a * t) / 2

PRINT luas

STOP

9. Bagaimana cara membuat flowchart untuk menghitung nilai A dan B


Jawab:

1. Menghitung Luas Segitiga

Algoritma Menghitung Luas Segitiga

Analisis :

Input : a (alas) dan t (tinggi)

Luas Segitiga = a*t/2

Algoritma :

Masukan nilai alas (a) dan nilai tinggi segitiga (t)

Maka untuk menghitung luas digunakan rumus alas dengan tinggi yang sudah ditentukan

Rumus untuk menghitung Luas Segitiga yaitu L = 1/2*a*t

Nilai L (Luas) akan dicetak sebagai output ke perangkat output (keluaran)

Algoritma Keliling Lingkaran

Analisis :

Input : r (jari-jari lingkaran) dan phi

Keliling Lingkaran  K=2*phi*r

Algoritma :

Menetapkan nilai phi = 3.14

Menghitung jari-jari lingkaran

Rumus untuk menghitung Keliling Lingkaran  yaitu K=2*phi*r

Nilai K (Keliling Lingkaran ) akan dicetak sebagai output ke perangkat output (keluaran)

Algoritma Luas Persegi Panjang

Analisis :

Input : p (panjang) dan l (lebar)

Luas Persegi Panjang  L = p*l

Algoritma :

Inputkan panjang

Inputkan lebar

Rumus untuk menghitung L  yaitu L= p*l

Nilai  L (Luas ) akan dicetak sebagai output ke perangkat output (keluaran)
