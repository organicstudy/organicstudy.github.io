Jawaban:

25. KD 3.136. Tuliskan 3 pecahan yang senilai dengan2/7​


Jawaban:

3 Pecahan yang Senilai dengan 2/7 adalah :

1. 4/14 (Hasil dari 2/7 x 2/2)

2. 8/28 (Hasil dari 2/7 x 4/4)

3. 16/56 (Hasil dari 2/7 x 8/8).

Diket :

Terdapat pecahan 2/7

Dit :

Tentukan 3 pecahan senilai lainnya

Jawab :

Pecahan senilai pertama

2/7 = 2 x 2 / 7 x 2 = 4/14

Pecahan senilai kedua

2/7 = 2 x 3 / 7 x 3 = 6/21

Pecahan senilai ketiga

2/7 = 2 x 4 / 7 x 4 = 8/28

Sehingga, jawaban sesingkat singkatnya adalah 2/7 = 4/14 = 6/21 = 8/28
