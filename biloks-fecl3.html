Ketentuan yang digunakan untuk menentukan biloks yaitu :

Jumlah biloks dari unsur bebeas adalah 0.Jumlah seluruh biloks atom atom penyusun satu ion sama dengan muatan ion tersebut.Jumlah biloks H bernilai +1.Jumlah biloks O bernilai -2.Jumlah biloks unsur unsur pembentuk senyawa netral sama dengan nol.

e. Biloks Cl₂O₃ = (2 × biloks Cl) + (3 × biloks O)

                    0  = (2 × biloks Cl) + (3 × (-2))

                    0  = (2 × biloks Cl) + (-6)

  2 × biloks Cl  =  0 + 6

  2 × biloks Cl  = 6

        biloks Cl  = 6/2

        biloks Cl  = +3

f. Biloks HClO₃ = biloks H + biloks Cl + (3 × biloks O)

                      0 = (+1) + biloks Cl + (3 × (-2))

                      0 = 1 + biloks Cl - 6

                      0 = biloks Cl - 5

        biloks Cl   = +5

14. K2CrO7, berapa bilangan biloks Cr? CaSO4, berapa biloks S? NaNO3 , berapa biloks N ? NH4+ Berapa biloks N ? SO2, berapa biloks S?


1)  Bilangan biloks Cr dalam K₂CrO₇ adalah +12.

2) Bilangan biloks S dalam CaSO₄ adalah +6.

3) Bilangan biloks N dalam NaNO₃ adalah +5.

4) Bilangan biloks N dalam NH₄⁺ adalah -3.

5) Bilangan biloks S dalam SO₂ adalah +4.

Pembahasan : 1)  Bilangan biloks Cr dalam K₂CrO₇

biloks K = +1 (Golongan IA)

aturan biloks :  

Bilangan oksidasi unsur golongan utama (IA,IIA,IIIA,IVA,VA,VIA ) sesuai dengan golongannya. Contoh : IA = +1, IIA = +2, IIIA = +3, IVA = -4, VA = -3 , VIA = -2 , VIIA = -1 , VIIIA = 0.

biloks O = -2  

aturan biloks :  

Bilangan oksidasi dalam senyawa O adalah -2 kecuali dalam peroksida (-1) dan dalam senyawa biner dengan fluor (+2). Contoh : Bilangan oksidasi O dalam H₂O = -2 , kecuali bilangan oksidasi O dalam H₂O₂ dan BaO₂ = -1 , dan bilangan oksidasi O dalam OF = +2.

biloks K₂CrO₇ = 0  

(2 × biloks K) + biloks Cr + (7 × biloks O) = 0

(2 × (+1)) + biloks Cr + (7 × (-2)) = 0

(+2) + biloks Cr + (-14) = 0

biloks Cr + (-12) = 0

biloks Cr = +12

2) Bilangan biloks S dalam CaSO₄

biloks Ca = +2 (Golongan IIA)

aturan biloks :  

Bilangan oksidasi unsur golongan utama (IA,IIA,IIIA,IVA,VA,VIA ) sesuai dengan golongannya. Contoh : IA = +1, IIA = +2, IIIA = +3, IVA = -4, VA = -3 , VIA = -2 , VIIA = -1 , VIIIA = 0.

biloks O = -2  

aturan biloks :  

Bilangan oksidasi dalam senyawa O adalah -2 kecuali dalam peroksida (-1) dan dalam senyawa biner dengan fluor (+2). Contoh : Bilangan oksidasi O dalam H₂O = -2 , kecuali bilangan oksidasi O dalam H₂O₂ dan BaO₂ = -1 , dan bilangan oksidasi O dalam OF = +2.

biloks CaSO₄ = 0  

biloks Ca + biloks S + (4 × biloks O) = 0

(+2) + biloks S + (4 × (-2)) = 0

(+2) + biloks S + (-8) = 0

biloks S + (-6) = 0

biloks S = +6

3) Bilangan biloks N dalam NaNO₃

biloks Na = +1 (Golongan IA)

aturan biloks :  

Bilangan oksidasi unsur golongan utama (IA,IIA,IIIA,IVA,VA,VIA ) sesuai dengan golongannya. Contoh : IA = +1, IIA = +2, IIIA = +3, IVA = -4, VA = -3 , VIA = -2 , VIIA = -1 , VIIIA = 0.

biloks O = -2  

aturan biloks :  

Bilangan oksidasi dalam senyawa O adalah -2 kecuali dalam peroksida (-1) dan dalam senyawa biner dengan fluor (+2). Contoh : Bilangan oksidasi O dalam H₂O = -2 , kecuali bilangan oksidasi O dalam H₂O₂ dan BaO₂ = -1 , dan bilangan oksidasi O dalam OF = +2.

biloks NaNO₃ = 0  

biloks Na + biloks N + (3 × biloks O) = 0

(+1) + biloks N + (3 × (-2)) = 0

(+1) + biloks N + (-6) = 0

biloks N + (-5) = 0

biloks N = +5

4) Bilangan biloks N dalam NH₄⁺

biloks H = +1

aturan biloks :  

Bilangan oksidasi hidrogen dalam senyawanya adalah +1 kecuali dalam Hidrida, atom Hidrogen biloksnya adalah -1. Contoh : Bilangan oksidasi H dalam H₂O, NH₃, dan HCl = +1, kecuali Bilangan oksidasi H dalam NaH dan CaH₂ = -1.

biloks NH₄⁺ = +1

biloks N + (4 × biloks H)= +1

biloks N + (4 × (+1)) = +1

biloks N + (+4) = +1

biloks N + (+4) = +1

biloks N = (+1) + (-4)

biloks N = -3

5) Bilangan biloks S dalam SO₂

Biloks O = -2  

aturan biloks :  

Bilangan oksidasi dalam senyawa O adalah -2 kecuali dalam peroksida (-1) dan dalam senyawa biner dengan fluor (+2). Contoh : Bilangan oksidasi O dalam H₂O = -2 , kecuali bilangan oksidasi O dalam H₂O₂ dan BaO₂ = -1 , dan bilangan oksidasi O dalam OF = +2.

biloks SO₂ = 0  

biloks S + (2 × biloks O) = 0

biloks S + (2 × (-2)) = 0

biloks S + (-4) = 0

biloks S = +4

Bilangan oksidasi dalam senyawa O adalah -2 kecuali dalam peroksida (-1) dan dalam senyawa biner dengan fluor (+2). Contoh : Bilangan oksidasi O dalam H₂O = -2 , kecuali bilangan oksidasi O dalam H₂O₂ dan BaO₂ = -1 , dan bilangan oksidasi O dalam OF = +2.

biloks Na₂SO₄ = 0  

(2 × biloks Na) + biloks S + (4 × biloks O) = 0  

(2 × (+1)) + biloks S + (4 × (-2)) = 0  

(+2) + biloks S + (-8) = 0  

biloks S + (-6) = 0  

biloks S = +6

7. Biloks H pada Na₂H₂

biloks Na = +1 (merupakan unsur golongan IA)  

aturan biloks :  

Bilangan oksidasi unsur golongan utama (IA,IIA,IIIA,IVA,VA,VIA) sesuai dengan golongannya. Contoh : IA = +1, IIA = +2, IIIA = +3, IVA = -4, VA = -3 , VIA = -2 , VIIA = -1 , VIIIA = 0.  

biloks Na₂H₂ = 0  

(2 × biloks Na) + (2 × biloks H) = 0  

(2 × (+1)) + (2 × biloks H) = 0  

(+2) + (2 × biloks H) = 0  

(2 × biloks H) = -2

biloks H = [tex]\frac{-2}{2}[/tex]

biloks H = -1

19. biloks dari 2nah2po3 ?biloks dari 2ph3 ?​


Jawaban:

Biloks dari 2NaH2PO3 (natrium hipofosfit) adalah:

- Biloks natrium (Na) adalah +1 karena ia adalah logam alkali.

- Biloks hidrogen (H) adalah +1 karena ia umumnya memiliki bilangan oksidasi positif ketika terikat dengan nonlogam.

- Biloks fosfor (P) adalah +1 karena ada dua hidrogen yang terikat ke fosfor dan masing-masing hidrogen memiliki bilangan oksidasi +1. Namun, ada dua atom oksigen di sebelah fosfor, yang masing-masing memiliki bilangan oksidasi -2. Karena total muatan dari senyawa harus netral, maka bilangan oksidasi dari fosfor harus +1 untuk menetralkan total bilangan oksidasi hidrogen dan oksigen. Oleh karena itu, bilangan oksidasi fosfor dalam 2NaH2PO3 adalah +1.

Jadi, biloks dari 2NaH2PO3 adalah: Na(+1) + 2H(+1) + P(+1) + 3O(-2) = 0.

Sedangkan, biloks dari 2PH3 (difosfina) adalah:

Biloks fosfor (P) adalah -3 karena ia terikat dengan tiga atom hidrogen (H), dan masing-masing hidrogen memiliki bilangan oksidasi +1. Oleh karena itu, untuk menetralkan total bilangan oksidasi hidrogen dan fosfor harus -3.

Jadi, biloks dari 2PH3 adalah: 2P(-3) + 6H(+1) = 0.

27. berapa gram sampel FeCl3 yang ditimbang bila diperlukan sebanyak 0,254 mol FeCl3 dalam suatu reaksi kimia


Massa sampel FeCl₃ yang ditimbang bila diperlukan sebanyak 0,254 mol FeCl3 dalam suatu reaksi kimia adalah 41,275 gram

Pembahasan

Dalam bahasa Yunani, kata stoicheion berarti unsur sedangkan stoikiometri (stoichiometry) berarti mengukur unsur meliputi massa atom dan massa rumus, rumus kimia, persamaan kimia serta semua yang berhubungan dengan reaksi kimia. Dalam stoikiometri, satuan yang digunakan untuk menyatakan jumlah partikel dalam zat dinamakan mol.

Cara menentukan jumlah mol :

1. Menentukan dengan jumlah partikel

Standar yang digunakan untuk menyatakan 1 mol adalah jumlah partikel dalam isotop atom C-12 (karbon) bermassa 12 gram.

12 gram C-12 mengandung 6,02 x 1023 partikel / atom / molekul / ion

Jumlah partikel tersebut dikenal dengan bilangan Avogadro (L).

bilangan Avogadro (L) = 6,02 x 1023 partikel / atom / molekul / ion

2. Menentukan jumlah mol dengan volume

a. Jika pengukuran dilakukan pada suhu 0oC dan tekanan 1 atm (STP (Standard Temperature and Pressure)), maka

n = Vm/22,4

dengan, Vm = volume molar standar

b. Menggunakan hipotesis Avogadro

pada suhu dan tekanan yang sama, gas-gas yang memiliki volume sama akan memiliki jumlah molekul yang sama.

V1 : V2 = n1 : n2


c. Jika pengukuran dilakukukan tidak pada suhu 0oC dan tekanan 1 atm, menggunakan hukum gas ideal :

PV = nRT

n = PV/RT

dengan:

P = tekanan (atmosfer)

T = suhu mutlak (Kelvin), K = oC + 273

V = volume (liter)

n = jumlah mol (mol)

R = tetapan gas ideal yang nilainya 0,082 L atm mol-1 K-1


3. Menentukan dengan massa molekul relatif

n = gram/Mr

dengan,

n = jumlah mol (mol)

gram = massa (gram)

Mr = massa molekul relatif


Diket :

Ar Fe = 56

Ar Cl = 35,5

n FeCl₃ = 0,254 mol


Ditanya :

massa  FeCl₃


Jawab :

Menentukan Mr FeCl₃

Mr FeCl₃  = 1. Ar Fe + 3. Ar Cl

Mr FeCl₃  = 1. 56 + 3. 35,5

Mr FeCl₃  = 56 + 106,5

Mr FeCl₃  = 162,5

Menentukan massa sampel FeCl₃

n = gram/Mr maka,

n FeCl₃ = [tex]\frac{massa}{Mr}[/tex]

0,254 = [tex]\frac{massa}{162,5}[/tex]

massa FeCl₃  = 0,254 x 162,5

massa FeCl₃  = 41,275 gram

maka, massa sampel FeCl₃ yang ditimbang adalah 41,275 gram
