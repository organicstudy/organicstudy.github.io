8. Menyelesaikan Masalah kontekstual dengan SPLTV.Tolong dibantu ya kak, terimakasih​


Jawab:

Bilangan itu adalah 349.Harga per kg salak = Rp6.000,00, harga per kg jeruk = Rp4.000.00, dan harga per kg apel = Rp7.500,00.Ketiga bilangan tersebut adalah 12, 14, dan 22.Bilangan tersebut adalah 126.

Penjelasan dengan langkah-langkah:

Nomor 1

Jika digit pembentuk bilangan itu adalah x, y, dan z, maka:

x + y + z = 16        ....(1)

x + y = z - 2          ....(2)

100x + 10y + z = 21(16) + 13

100x + 10y + z = 336 + 13

100x + 10y + z = 349   ....(3)

Kesimpulan sementara:

Bilangan tersebut adalah 349, karena 3+4 = 9-2 dan 3+4+9 = 16

Substitusi (2) ke (1):

x + y + z = 16

z - 2 + z = 16

2z = 18

z = 9

Persamaan (2) menjadi:

x + y = 7     ....(4)

Persamaan (3) menjadi:

100x + 10y = 340

10x + y = 34   ...,(5)

Dari persamaan (4) kita tahu bahwa:

y = 7 - x

Maka:

10x + 7 - x = 34

9x = 27

x = 3

Oleh karena itu:

y = 7 - 3 = 4

∴ Kesimpulan akhir: Bilangan tersebut adalah 349.

Nomor 2

Harga per kg salak = x

Harga per kg jeruk = y

Harga per kg apel = z

x + 3y + 2z = 33.000     ....(1)

2x + y + z = 23.500       ....(2)

x + 2y + 3z = 36.500     ....(3)

Dari persamaan (1) dan (3):

x + 2y + 3z = 36.500

x + 3y + 2z = 33.000

----------------------------- -

-y + z = 3.500

y = z - 3.500      ....(4)

Substitusi nilai y ke persamaan (2):

2x + z - 3.500 + z = 23.500

2x + 2z = 27.000

x + z = 13.500

x = 13.500 - z    ....(5)

Substitusi nilai x dan y ke persamaan (3):

13.500 - z + 2(z - 3.500) + 3z = 36.500

-z + 2z + 3z + 13.500 - 7000 = 36.500

4z + 6.500 = 36.500

4z = 30.000

z = 7.500

x = 13.500 - z

x = 13.500 - 7.500

x = 6.000

y = z - 3.500

y = 7.500 - 3.500

y = 4.000

∴ Jadi, harga per kg salak = Rp6.000,00, harga per kg jeruk = Rp4.000.00, dan harga per kg apel = Rp7.500,00.

 

Nomor 3

(a + b + c) / 3 = 16

a + b + c = 48   ...(1)

b + 20 = a + c

a - b + c = 20   ...(2)

c = a + b - 4

a + b - c = 4    ...(3)

Eliminasi b dan c dengan (2) + (3):

a - b + c = 20

a + b - c = 4

------------------- +

2a = 24

a = 12

Persamaan (1) menjadi:

12 + b + c = 48

b + c = 36    ...(4)

Persamaan (2) menjadi:

12 - b + c = 20

-b + c = 8    ...(5)

Persamaan (3) menjadi:

12 + b - c = 4

b - c = -8   ekuivalen dengan persamaan (5)

Eliminasi c dengan (4) - (5):

b + c = 36

-b + c = 8

---------- -

2b = 28

b = 14

Substitusi nilai b untuk mendapatkan c:

14 + c = 36

c = 22

∴ Jadi, ketiga bilangan tersebut adalah 12, 14, dan 22.

Nomor 4

x + y + z = 9         ....(1)

100x + 10y + z = 126        ....(2)

z - x - y = 3          ....(3)

-x - y + z = 3        ....(4)

Eliminasi z:

x + y + z = 9

-x - y + z = 3

-------------- -

2x + 2y = 6

x + y = 3     ....(5)

Substitusi nilai (x + y) ke persamaan (3):

z - x - y = 3

z - (x + y) = 3

z - 3 = 3

z = 6

Maka persamaan (2) menjadi:

100x + 10y + 6 = 126

100x + 10y = 120

10x + y = 12

9x + (x + y) = 12

9x + 3 = 12    ....(substitusi nilai (x+y) dari persamaan (5))

9x = 9

x = 1

Karena x + y = 3, maka:

y = 3 - x

y = 2

∴ Jadi, bilangan tersebut adalah 126

Pertanyaan :

Andi, Bayu & Jodi panen jeruk. Hasil kebun Jodi 10 kg lebih sedikit dari hasil kebun Andi dan hasil panen jeruk Jodi lebih banyak 10 kg dari hasil kebun Bayu. Jika hasil panen ketiganya adalah 195 kg, maka tentukanlah hasil panen jeruk masing-masing

Jawab :

Jika

- Andi = a

- Bayu = b

- Jodi = j

Buat menjadi bentuk SPLTV

j = a - 10  ...(i)

j = b + 10   ...(ii)

a + b + j = 195   ...(iii)

Subtitusikan persamaan (i) ke dalam (iii)

a+b+j = 195

(a - 10) + b + j = 195

2a + b = 205    .... (iv)

Subtitusikan persamaan (ii) ke dalam (iii)

a+b+j = 195

a + b + (b + 10) = 195

a + 2b = 185     .... (v)

Eliminasikan persamaan (iv) dan (v)

2a + b = 205  |x1|   2a + b = 205

a + 2b = 185   |x2|   2a + 4b = 370

                               ___________ _

                                      -3b = -165

                                          b = -165/-3

                                          b = 55

Subtitusikan nilai b ke dalam persamaan (ii)

j = b + 10

j = 55 + 10

j = 65

Subtitusikan nilai b dan j ke dalam persamaan (iii)

a + b + j = 195

a + 55 + 65 = 195

a + 120 = 195

a = 195 - 120

a = 75

Jadi, hasil panen jeruk Andi, Bayu, dan Jodi berturut-turut adalah 75 kg, 55 kg, dan 65 kg

29. Apa saja macam masalah sosial kontekstual?


Pembelajaran kontekstual (Contextual Teachingand Learning) adalah konsep belajar di mana guru menghadirkan dunia nyata ke dalam kelas dan mendorong siswa membuat hubungan antara pengetahuan yang dimilikinya dengan penerapannya dalam kehidupan mereka sehari-hari.
dalam kehidupan mereka sehari-hari, dengan melibatkantujuh komponen, yakni:1.a.Kontruktivisme (Constuctivism)
Kontruktivisme (contructivism) merupakan landasan berpikir (filosofi) pembelajaran kontekstual, yaitu bahwa pengetahuan dibangun oleh manusia sedikit demi sedikit, yang hasilnya diperluas melalui konteks yang terbatas (sempit).
1.b.Bertanya(Questioning)
Bertanya (questioning) adalah suatu strategi yang digunakan secara aktif oleh siswa untuk menganalisis dan mengeksplorasi gagasan-gagasan.
1.c.Menemukan(Inquiri)
Menemukan (inquiry) merupakan bagian inti dari kegiatan pembelajaran berbasis kontekstual
1.d.Pemodelan(modeling) yaitu dalam sebuah pembelajaran keterampilan atau pengetahuan tertentu, ada model yang bisa ditiru.
1.e.Masyarakat belajar (Learning Community)
Masyarakat belajar (learning community), hasil pembelajaran diperoleh dari kerjasama dengan orang lain.
1.f.Refleksi(Reflection)
Refleksi (reflection) adalah cara berpikir tentang apa yang baru dipelajari atau berpikir ke belakang tentang apa-apa yang sudah kita lakukan di masa yang lalu
1.g.Penilaian sebenarnya(Authentic Assessment)
Penilaian yang sebenarnya (authentic assessement), merupakan prosedur penilaian pada pembelajaran kontekstual yang memberikan gambaran perkembangan belajar siswanya. Assessement adalah proses pengumpulan berbagai data yang bisa memberikan gambaran perkembangan belajar siswa.

Dalam pengajaran kontekstual memungkinkan terjadinya lima bentuk belajar yang penting, yaitu mengaitkan (relating), mengalami(experiencing),menerapkan (applying),bekerjasama(cooperating) dan mentransfer(transferring).
