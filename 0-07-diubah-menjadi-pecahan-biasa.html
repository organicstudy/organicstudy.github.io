Caranya adalah dengan memastikan terlebih dahulu bahwa dalam windows 7 tersebut sudah ada program Ms. Word 2016:

1. Bukalah program Ms. Word 2016

2. Klik menu File dan klik Open

25. Diantara bilangan berikut yang berada diantara 0 06 dan 0 07 adalah


Jawaban:

0,065

Penjelasan dengan langkah-langkah:

0,06 = 6/100

0,07 = 7/100

(6 x 2)/(100 x 2)

= 12/200

(7 x 2)/(100 x 2)

= 14/200

bilangan antara 12/200 dan 14/200 adalah 13/200

13/200

= 0,065

jadi, bilangan antara 0,06 dan 0,07 adalah 0,065
