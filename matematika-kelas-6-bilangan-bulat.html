16. Tuliskan bilangan bulat negatif dalam kehidupan sehari hariMapel: MatematikaMateri:Bilangan Bulat Tema:1 Selamatkan makhluk hidupKelas:6 SD​


Contoh bilangan negatif dalam kehidupan sehari-hari

Roni berjalan mundur 5 langkah. Penggali sumur, menggali sumur sedalam 3 meter. Nita menyelam sedalam 4 meter dari permukaan laut.

[tex]Pembahasan[/tex]

Bilangan bulat adalah bilangan yang terdiri dari bilangan negatif, nol dan bilangan positif, bilangan bulat dilambangkan dengan B.

Bilangan bulat ada 2, bilangan bulat positif dan bilangan bulat negatif.

Antara bilangan bulat positif dan bilangan bulat negatif dipisahkan oleh bilangan netral yaitu 0.

Bilangan disebelah kanan 0 atau disebelah atas 0 adalah bilangan bulat positif, dan bilangan disebelah kiri 0 atau disebelah bawah 0 adalah bilangan bulat negatif.

✎ Lawan dari bilangan Negatif adalah bilangan Positif. ✎

Kegiatan yang termasuk bilangan positif jika ditulis dengan lambang bilangan yaitu, Untung, Maju, Kekanan, Keatas, Diatas, Naik, Sebelum.

Kegiatan yang termasuk bilangan negatif jika ditulis dengan lambang bilangan yaitu, Rugi, Mundur, Kekiri, Kebawah, Dibawah, Turun, Sesudah.

24. Buktikan bahwa 2+6+10...+ (4n-2)=2n kuadrat. Berlaku untuk semua n bilangan bulat! Thx, tolong ya Matematika wajib kelas xi


2+6+10+...+(4n-2) = 2n²
Langkah 1 :
untuk n=1
(4n-2) =2n²
(4(1)-2) = 2(1)²
(4-2) = 2
2 = 2 (benar untuk n = 1)
Langkah 2:
untuk n = k
2+6+10+...+(4k-2) = 2k²
untuk n = k+1
*ruas kanan
2+6+10+...+(4k-2)+(4(k+1)-2)
2k² + 4k + 4 - 2
2k² + 4k + 2
*ruas kiri
2n²
=(2(k+1))²
=4k² + 4k + 1
