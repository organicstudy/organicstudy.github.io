1. P ∨ (P ∧ (¬P ∨ Q)) ∨ Q ≡ P ∨ Q

Untuk membuktikan bahwa kedua pernyataan tersebut equivalen, kita dapat menggunakan hukum kesamaan logika sebagai berikut:

P ∨ (P ∧ (¬P ∨ Q)) ∨ Q ≡ P ∨ ((P ∧ ¬P) ∨ (P ∧ Q)) ∨ Q (hukum komplementer)≡ P ∨ (F ∨ (P ∧ Q)) ∨ Q (hukum konjungsi)≡ P ∨ (P ∧ Q) ∨ Q (hukum tertinggal)≡ P ∨ Q (hukum disjunsi)

2. (P ∧ ¬Q) ∨ (¬P ∧ Q) ≡ ¬(P ∧ Q) ∧ (P ∨ Q)

Untuk membuktikan bahwa kedua pernyataan tersebut equivalen, kita dapat menggunakan hukum kesamaan logika sebagai berikut:

Untuk membuktikan bahwa kedua pernyataan tersebut equivalen, kita dapat menggunakan hukum kesamaan logika sebagai berikut:

4. ¬(P ↔ Q) ≡ ¬P ↔ Q
