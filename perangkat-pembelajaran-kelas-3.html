(+) Subject + Verb 1 with s/es + Object + evidence (time, place, etc)

(-) Subject + Do/does + not + Verb 1 + Object + evidence (time, place, etc)

(?) Do/does + Subject + Verb 1 + Object + evidence (time, place, etc)
