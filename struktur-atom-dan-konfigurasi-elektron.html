9. Konfigurasi elektron atom


Jawaban:

Konfigurasi elektron atom adalah susunan orbital-orbital elektron dalam sebuah atom. Orbit-orbit elektron ini ditempatkan dalam serangkaian kulit atau level energi yang berbeda. Setiap kulit dapat menampung sejumlah maksimum elektron, yang ditentukan oleh rumus 2n^2, di mana n adalah nomor kulit.

Kita dapat menuliskan konfigurasi elektron atom menggunakan diagram kulit elektron atau notasi orbital. Diagram kulit elektron menunjukkan jumlah elektron dalam setiap kulit, sedangkan notasi orbital menunjukkan jumlah elektron dalam setiap subkulit (orbital).

Sebagai contoh, konfigurasi elektron atom karbon (C) adalah sebagai berikut:

- Diagram kulit elektron: 2, 4

- Notasi orbital: 1s^2 2s^2 2p^2

Ini berarti karbon memiliki dua elektron di kulit pertama (level energi 1) dan empat elektron di kulit kedua (level energi 2). Dalam notasi orbital, kita menuliskan konfigurasi elektron sebagai susunan dari orbital-orbital pada masing-masing kulit. Jadi, karbon memiliki 2 elektron di subkulit 1s, 2 elektron di subkulit 2s, dan 2 elektron lainnya di salah satu dari tiga orbital subkulit 2p.

Konfigurasi elektron atom dapat membantu memahami sifat atom seperti kestabilannya, ketidakstabilannya, reaktivitasnya, serta periode dan golongan di dalam tabel periodik unsur.

19. Tentukan keempat bilangan kuantum dari atom oksigen dengan nomor atom 8 berdasarkan struktur konfigurasi elektron jelaskan


Jawab :

Tentukan keempat bilangan kuantum dari atom oksigen dengan nomor atom 8 berdasarkan struktur konfigurasi elektron : n = 2 ; l = 1 ; m = -1 ; s =  -½

⇒Kesimpulan, Tentukan keempat bilangan kuantum dari atom oksigen dengan nomor atom 8 berdasarkan struktur konfigurasi elektron : n = 2 ; l = 1 ; m = -1 ; s =  -½

Pembahasan :

KONFIGURASI ELEKTRON

Ada 3 aturan penulisan konfigurasi elektron :

1.  Aturan Aufbau

Aufbau berprinsip bahwa pengisian elektron pada orbital dimulai dari tingkat energi terendah ke tingkat energi lebih tinggi.

Urutan pengisian berdasarkan tingkat energi :

1s² 2s² 2p⁶ 3s² 3p⁶ 4s² 3d¹⁰ 4p⁶ 5s² 4d¹⁰ 5p⁶ 6s² 4f¹⁴ 5d¹⁰ 6p⁶ 7s² 5f¹⁴ 6d¹⁰ 7p⁶

Contoh :

Konfigurasi elektron :

₃Li = 1s² 2s¹

₁₁Na = 1s² 2s² 2p⁶ 3s¹

₂₁Sc : 1s² 2s² 2p⁶ 3s² 3p⁶ 4s² 3d¹

2. Aturan Hund

Yaitu pengisian elektron pada orbital-orbital yang tingkat energinya sama elektron tidak berpasangan sebelum orbital-orbital lainnya masing-masing terisi satu elektron.

Asas ini dikemukakan berdasarkan penalaran bahwa energi tolak-menolak antara dua elektron akan minimum jika jarak antara elektron berjauhan.

Contoh :

Tentukan diagram orbital untuk unsur-unsur berikut :

₇N = 1s²   2s²        2p³

       ⇅      ⇅      ↑    ↑    ↑

₉F = 1s²    2s²            2p⁵

       ⇅      ⇅         ⇅    ⇅     ↑

3.  Larangan Pauli

Pauli mengemukakan hipotesisnya yang menyatakan bahwa dalam satu atom tidak mungkin dua elektron mempunyai keempat bilangan kuantum sama. Misal, 2 elektron akan menempati subkulit 1s. Tiga bilangan kuantum pertama akan mempunyai nilai yang sama (n = 1, l = 0, m = 0). Untuk itu bilangan kuantum yang terakhir, yaitu bilangan kuantum spin (s) harus mempunyai nilai berbeda (+½ atau -½) . Dengan kata lain, setiap orbital maksimal hanya dapat terisi 2 elektron dengan arah spin berlawanan.

Jumlah elektron maksimal untuk tiap subkulit sama dengan dua kali dari jumlah orbitalnya.

orbital s maksimal 2 elektron,

orbital p maksimal 6 elektron,

orbital d maksimal 10 elektron, dan

orbital f maksimal 14 elektron.

Contoh :

Tentukan harga keempat bilangan kuantum pada elektron terakhir 2s¹.

2s¹  →   n = 2 ; l = 0 ; m = 0 ; s = +½

Tentukan harga keempat bilangan kuantum pada :

₂₀Ca = 1s² 2s² 2p⁶ 3s² 3p⁶ 4s²

4s²  →   n = 4 ; l = 0 ; m = 0 ; s = -½
